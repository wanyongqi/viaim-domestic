## 二、各耳机通用设置功能

以下功能所有型号均支持（除特殊说明外）：

| 功能 | 说明 |
|------|------|
| 左右耳电量展示 | 展示当前左右耳电量 |
| 蓝牙连接管理 | 查看当前连接设备；断开连接；连接历史设备；连接新设备；删除配对关系 |
| 手势控制 | 双耳在线时点击弹出「是否更新耳机手势？」弹窗；详细规则见 `bluetooth-gesture.md` |
| 耳机音量 | 可调节耳机音量（Pro3/Pro2/Air2 支持）|
| 查找设备 | 通过播放声音查找耳机 |
| 固件升级 | 展示固件版本；有新固件时可 OTA 升级 |
| 关于设备 | 展示名称/型号/固件版本/序列号；可切换查看左耳/右耳/盒子信息 |
| 开盖显示电量（Android）| 开启：开盖时显示耳机电量；关闭：手动查看 |

### 蓝牙连接管理详细规则

**设备列表规则：**
- 按「最近成功连接时间」倒序排列
- 最多同时保留 2 个连接设备；连接第 3 个设备时，断开最老且非发声的设备
- 蓝牙连接管理页顶部展示「连接 Windows 设备使用注意事项」入口

**断开/删除设备操作规则：**

| 操作 | 设备状态 | 行为 |
|------|---------|------|
| 点击已连接设备 | 非发声状态 | 直接断开连接 |
| 点击已连接设备 | 发声状态 | 弹出二次确认断开弹窗；确认后断连 |
| 删除（移除配对）已连接设备 | — | 弹出移除耳机配对二次确认弹窗；确认后断连并删除 |
| 删除未连接设备 | — | 弹出移除配对弹窗；确认后删除 |
| 点击未连接设备 | — | 尝试重新连接；连接失败弹出连接失败弹窗 |

### 查找设备规则

- 可分别播放左耳/右耳，或双耳同时播放（同时发出嗡鸣声）
- 点击播放按钮（无论耳机是否佩戴）→ 弹出**音量警告弹窗**
- 弹窗点击「播放」→ 对应耳机发出嗡鸣声
- 点击关闭/取消/弹窗外 → 弹窗关闭，不播放

### 固件升级详细规则

**有新固件时的入口：**
- 首页左上角显示固件升级提示
- 耳机设置页显示固件升级提示栏；「固件更新」列表项显示红点

**升级前提条件：**

| 条件 | 行为 |
|------|------|
| 单/双耳未入盒 | 弹出「请将双耳放入充电盒」提示弹窗 |
| 任一组件电量 < 30% | 弹出「请连接电源」提示弹窗 |
| 已入盒 + 电量充足 | 可直接开始升级 |

**升级中断情况（均导致升级失败）：**
- 手机锁屏或返回系统首页 → 升级失败
- 杀掉 App → 升级失败；再次打开 App 连接耳机正常
- 耳机关盖 → 升级失败；可点击「重试」或「以后再说」

#### 3.5.2 语音控制设置（国内版）

**入口：** 耳机设置页 → 高级设置 → 语音控制

**页面结构（参考 Pro3）：**
- 顶部「**语音控制**」总开关（关闭后所有指令无效）
- 「**语言**」选择入口：支持中文等多种语言切换；下方提示「**万姆语音助手唤醒词：「你好，小万」；唤醒词会根据语言设置自动切换**」
- 指令列表按场景分组展示

#### 3.5.3 中文语音指令清单（国内版）

> 国内版语音指令为**中文**（海外版为英文）；语言切换后指令文案同步变化。

| 场景 | 指令示例 |
|------|---------|
| **播放音乐时** - 播放/暂停 | 「**播放音乐**」 / 「**暂停播放**」 |
| **播放音乐时** - 切换曲目 | 「**上一首**」 / 「**下一首**」 |
| **播放音乐时** - 调节音量 | 「**增大音量**」 / 「**减小音量**」 |
| **来电时** - 接听电话 | 「**接听电话**」 |
| **来电时** - 拒接电话 | 「**挂掉电话**」 |
| **切换降噪时** | 「**降噪模式**」 / 「**通透模式**」（仅 Pro3/Pro2 支持降噪切换；Air2 无此分组）|
| **录音时** - 操控录音 | 「**开始录音**」 / 「**停止录音**」 |

> **重要约束**：
> - 「**停止录音**」仅在**现场录音**时可用，**通话录音暂不支持**
> - 关闭语音控制后所有指令无效
> - Air2 无降噪控制，因此无降噪切换指令

#### 3.5.4 语音唤醒万姆（Pro3/Air2 专属）

**入口：** 耳机设置页 → 高级设置 → 「**语音唤醒万姆**」开关
- 开关说明：「说「你好，小万」，可以唤醒万姆，并进行对话」
- 界面语言切换时唤醒词同步变化：
  - 中文 → 「**你好，小万**」
  - English → 「**Hey Vitana**」（⚠️ 具体措辞待确认）
- 唤醒后进入**万姆/Vitana AI 助理半屏浮层**（详见 `wanmu-vitana.md` §二、§三）
- ⚠️ 待确认：
  - 开关默认状态（默认开启 or 关闭）
  - Pro2/Nano+/VIAIM Air 设置页是否完全不展示此开关项（型号降级隐藏）
  - 未佩戴耳机时喊唤醒词是否有反应

## 四、Kit（桌面端录音配件）

### 4.1 Kit 连接电脑

- 直接将 Kit 插入电脑端口
- Kit **白灯常亮**表示连接成功
- 连接成功后可作为电脑麦克风使用

### 4.2 Kit 与耳机配对

- 让 Kit 和耳机**同时进入配对状态**，再将两个设备靠近，自动完成配对
- Kit 进入配对的两种方式：
  1. 长按 Kit 侧边按键约 3 秒
  2. Kit 连接电脑时，在桌面端首页点击「连接耳机」按钮
- Kit 白灯**闪烁**表示进入配对状态

### 4.3 Kit 录音规则

- Kit 录音**必须通过桌面端**，不支持移动端
- Kit 支持独立连接电脑录音，或连接 viaim 耳机录音
- Kit 仅支持连接 viaim 耳机录音，连接其他品牌耳机**无法启动录音**

### 4.4 Kit 按键功能

- 通话中**短按** Kit 侧边按键：关闭/打开麦克风（麦克风关闭时**红灯常亮**）
- 可在桌面端 App 中将此按键功能改为**结束录音**

## 五、测试关注点

- 各耳机型号设置页功能差异（按型号逐一对照上表验证）
- VIAIM Air：不支持闪录、不支持噪声控制、不支持声音风格
- Nano+ / VIAIM Air：名称不可修改
- 音质偏好「声音质量优先」与双设备连接/音视频录音的互斥逻辑（仅 Pro3/Air2/Pro2）
- 佩戴检测：Air2/VIAIM Air 不支持
- 双设备连接：Nano+/VIAIM Air 默认支持无开关；Pro3/Air2/Pro2 有开关控制
- 固件升级前提：双耳入盒 + 电量 ≥ 30%
- 查找设备：无论耳机是否佩戴，点击播放均弹出音量警告弹窗
- **语音控制**：Pro3/Air2/Pro2 支持；**指令为中文**（非英文）；「停止录音」仅现场录音可用，通话录音不支持
- **语音唤醒万姆**：仅 Pro3/Air2 设置页存在此开关；Pro2/Nano+/VIAIM Air 不展示
- **唤醒词随语言切换**：中文「你好，小万」↔ 英文「Hey Vitana」；唤醒后进入 AI 助理浮层（详见 `wanmu-vitana.md`）
- 手势控制详细规则见 `bluetooth-gesture.md`

# 「我的」使用设置功能规格（国内版 viaim App）

> 版本：V1.1.0 | 最后更新：2026-04

---

## 一、页面入口

底部导航栏第四个 Tab「我的」，包含账号信息展示及以下设置模块。

> ⚠️ 注意：**「云同步」开关**位于「我的」→「账号设置」中，详见 `account-settings.md`。

---

## 二、技术支持 / 帮助与反馈

| 功能项 | 说明 |
|--------|------|
| 在线客服 | 点击跳转国内客服入口（非 Fin.ai）|
| 访问帮助中心 | 跳转国内帮助中心页面 |
| 热门问题（服务端配置）| 展示后台配置的热门问题列表，语言与 App 当前界面语言一致；点击跳转对应 Web 详情页 |
| 复制日志链接 | 点击进入 Loading 状态（不可其他操作）→ 生成成功后 Toast 提示「已复制日志链接」|

## 三、录音设置

> 部分设置项带有「上云」「本地」「不上云」标注，说明该设置影响的处理链路。

| 设置项 | 显示条件 | 说明 |
|--------|---------|------|
| 闪录设置入口 | **连接支持闪录的耳机时显示** | 点击进入闪录设置页 |
| 录音实时转写（上云）| 始终显示 | 开启：录音过程中实时转写文本；关闭：只录音不转写 |
| 闪录同步后自动转写（上云）| **连接支持闪录的耳机时显示** | 开启：闪录下载后自动转写；关闭：不自动转写 |
| 转写语言（本地）| 始终显示 | 从 46 种转写语言中选择（32 外语 + 12 方言 + 2 民族语言），选择后录音时按所选语言转写；详见 `reference-data/viaim-domestic/languages.csv` |
| 自动区分说话人（上云）| 始终显示 | 开启：录音时自动按说话人区分转写文本；关闭：不区分 |
| 位置信息记录（上云）| 始终显示 | 开启：录音记录展示位置信息；关闭：位置信息为空 |
| 允许在桌面查看录音状态 | **Android 专属** | 开启：实时录音时可悬浮窗查看录音状态 |
| 自动开始录音（不上云）| 始终显示 | 开启：耳机收到通话信号时自动开启通话录音；关闭：有通话时不自动开启 |

### 4.1 Flash Record 触发行为规则

> 仅在连接支持闪录的耳机时，录音设置中才显示「闪录设置」入口。

**设置选项：通话-自动选择 / 通话-闪存模式 / 现场-闪存模式**

| 设置选项 | 有通话 + 已连接App | 有通话 + 未连接App | 无通话 + 已连接App | 无通话 + 未连接App |
|--------|-----------------|-----------------|-----------------|-----------------|
| 通话-自动选择 | 启动 **App 端通话录音** | 启动耳机本地通话闪录 | — | — |
| 通话-闪存模式 | 启动耳机本地通话闪录 | 启动耳机本地通话闪录 | — | — |
| 现场-闪存模式 | — | — | 启动耳机本地现场闪录 | 启动耳机本地现场闪录 |

## 四、AI 设置

| 设置项 | 说明 |
|--------|------|
| 职业身份 | 输入框最多 **50 个字符**，可正常修改保存 |
| 默认 AI 模型 | 设置同步至：个性化模版、模版库、重新生成摘要/待办/脑图 |
| 聊天机器人模型 | 设置同步至：万姆（中文界面）/ Vitana（英文界面）|
| 输出语言 | 选择后，智能标题、概览、摘要待办、脑图、AI 助理回复均按所选语言输出 |
| 摘要模版个性化开关 | 开启：显示自定义模板入口；关闭：按默认格式生成 |
| 自定义模板 | 展示自定义模版列表，支持创建/编辑/删除/查看/应用 |
| 智能标题 | 开启：根据内容生成概括性标题；关闭：不生成智能标题 |
| 智能标题包含日期 | 开启：标题含日期前缀（如"10-17 录音标题"）；关闭：纯概括性标题 |

### 4.1 AI 大模型列表

国内版可选模型：**viaim AI / DeepSeek V3 / DeepSeek R1**

> 国内版**无 Basic / Pro / Ultra 套餐层级**（海外订阅制概念不适用）。
> AI 助理入口/页面 UI 详见 `wanmu-vitana.md`。

### 4.2 摘要模版详细规则

**字符上限：**
- 模板名称：最多 **20 个字符**
- 提示词：最多 **2000 个字符**

**创建模板流程：**
1. 进入模板库弹窗 → 点击「创建」→ 弹出创建模板窗口
2. 填写模板名称（必填，否则保存按钮置灰）；提示词可选填
3. 保存成功 → 返回模板库弹窗，**新建模板自动选中**

**模板库弹窗操作：**
- 支持左右滑动浏览模板列表
- 点击模板卡片 → 弹出预览弹窗
- 选中模板后**不可取消选中**（只能切换到其他模板）
- AI 设置页：上次选中的模板**下次打开自动选中**
- 详情页模板库：**每次打开均为未选中状态**

**编辑/删除模板：**
- 编辑：带入现有名称和提示词
- 删除：弹出二次确认弹窗 → 确认后**自动选中下一个**；无其他模板则展示空模板库

**多端同步：**
- 创建/编辑/删除操作**多端实时同步**

## 五、翻译设置（仅国内）

### 5.1 语音嘴替

适用耳机：**iFLYBUDS Pro2 / Pro3 / Air2**

此功能可复刻用户声音，在面对面翻译等场景中，当用户说一种语言时，手机用此声音合成翻译后的语言。

| 操作 | 说明 |
|------|------|
| 创建嘴替 | 在安静环境下对着手机麦克风念一段提示文字（无需佩戴耳机）|
| 使用嘴替 | 创建后在面对面翻译中自动采用，无需额外操作 |

### 5.2 翻译设置项

| 设置项 | 说明 |
|--------|------|
| 朗读速度 | 设置翻译播报的速度 |
| 测试朗读 | 测试当前翻译播报效果 |
| 离线语音包 | 下载后翻译功能支持不联网使用；翻译速度提升但准确率下降；**当前仅支持中文-英文** |
| 通话翻译开关 | 开启后方可启动通话翻译功能；**开启时会同步关闭「自动开始录音」** |
| 自动通知 | 通话接通后自动向对方播报开场白；**默认开启**，支持关闭 |

## 六、权限管理

| 权限项 | 平台 | 说明 |
|--------|------|------|
| 定位权限 | iOS & Android | 点击可跳转去系统授权 |
| 悬浮窗/视窗权限 | **Android 专属** | 点击可跳转去系统授权；iOS 不显示此项 |

**权限状态说明：**
- 每项右侧显示当前状态：「已授权」/ 「去授权」
- 在系统设置中开启权限后返回 App，列表状态**自动更新**
- Android < 12：蓝牙连接时需弹出位置权限页面；Android ≥ 12：**不再弹出定位权限**

|--------|------|

## 八、语言设置

国内版 App **仅支持简体中文和英语**两种界面语言。

| 设置项 | 可选项 |
|--------|--------|
| App 界面语言 | 简体中文 / English |

**语言切换影响：**
- 切换为中文 → AI 助理名称显示「万姆」
- 切换为英文 → AI 助理名称显示「Vitana」
- 切换界面语言**不影响**已设置的 AI 输出语言

## 十、测试关注点

**录音设置：**
- 闪录设置和「闪录同步后自动转写」仅连接支持闪录耳机时显示
- 「允许在桌面查看录音状态」仅 Android 显示
- 自动开始录音开关**不跨设备同步**；其他录音设置均上云同步

**AI 设置：**
- 输出语言影响智能标题/概览/摘要待办/脑图/AI助理回复；切换界面语言或转写语言不影响
- AI 设置页打开模板库自动选中上次选中项；详情页每次打开均未选中
- 摘要模板名称上限 **20 字符**；提示词 **2000 字符**
- 国内版可选模型：viaim AI / DeepSeek V3 / DeepSeek R1

**翻译设置：**
- 语音嘴替仅支持 Pro2/Pro3/Air2
- 通话翻译开关开启时自动关闭「自动开始录音」，需注意联动
- 离线语音包当前仅支持中英互译
- 自动通知默认开启，验证关闭后对方是否不再收到开场白

**主题设置：**
- 国内版仅深色模式，无浅色/跟随系统选项

**语言设置：**
- **UI 界面语言**仅简体中文/English 两选项（注意与"转写语言 46 种"区分）
- 切换界面语言同步更新 AI 助理名称（万姆 ↔ Vitana）

**权限管理：**
- iOS 无悬浮窗权限项；Android 有定位/悬浮窗两项
- Android ≥ 12：连接耳机不弹出定位权限请求

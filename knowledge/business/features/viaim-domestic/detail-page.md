# 详情页面功能规格（国内版 viaim App）

> 版本：V1.0.0 | 最后更新：2026-04

---

## 一、详情页类型与内容差异

| 详情页类型 | 适用来源 | 主要内容 |
|-----------|---------|---------|
| **有转写的音频详情页** | 耳机录音（开启转写）/ 闪录下载（开启自动转写）| 音频播放 + 转写 + 摘要&待办 + 思维导图 + **AI 对话浮球（万姆/Vitana）**|
| **仅录音详情页** | 耳机录音（关闭转写）/ 闪录下载（未开启自动转写）| 仅音频播放，无转写和 AI 内容（⚠️ 待确认浮球是否展示）|

> 国内版 App **不支持上传音频/文档**，无文档详情页类型。
> **AI 浮球入口**：详情页右下角小万/Vitana 图标，点击弹出半屏 AI 助理浮层；详见 `wanmu-vitana.md` §二、§三。

---

## 二、标题与概览

| 功能 | 说明 |
|------|------|
| **智能标题** | 开启智能标题开关后，根据转写内容自动生成标题；适用于所有录音类型（通话/现场/音视频/实时翻译）|
| **修改标题** | 点击标题旁修改按钮弹出弹窗，可修改录音标题 |
| **概览** | 根据转写内容自动生成概览（精简摘要）|

---

## 三、导出功能

详见 `domain/export-rules.md`，详情页导出支持：

| 导出类型 | 格式 |
|----------|------|
| 导出音频 | 原音频文件 |
| 导出转写 | txt / docx / pdf（含时间戳/说话人选项）|
| 导出摘要 | txt / markdown / docx / pdf |
| 导出思维导图 | **jpeg / markdown / pdf**（详情页独有 jpeg）|

> 国内版无文档上传，无「导出原文件（文档）」选项。

## 四、分享链接（仅国内）

| 功能 | 说明 |
|------|------|
| 生成分享链接 | 可生成链接将记录内容共享给他人，对方在浏览器中查看转写和摘要 |
| 链接密码 | 支持设置密码保护内容安全 |
| 链接有效期 | 永久 / 7天 / 30天 / 90天，默认**永久** |
| 分享受限 | 若触发平台安全保护机制，分享功能将被限制；需提供账号信息由人工复审后解除 |

## 九、转写 Tab

| 功能 | 说明 |
|------|------|
| **编辑** | 点击编辑按钮进入编辑文本页，可编辑转写原文/译文 |
| **显示译文** | 开启/关闭译文显示 |
| **复制记录** | 仅复制文本 / 复制原文和翻译 |
| **重新转写** | 从 46 种转写语言中选择后重新转写（含方言与民族语言，见 `reference-data/viaim-domestic/languages.csv`）；受 §8.1 非实时转写次数上限约束 |
| **区分说话人** | 开启：选择说话人数（2~9/自动），按声纹分说话人；关闭：不区分 |
| **查找与替换** | 输入关键词查找高亮，可替换当前或全部关键词 |
| **修改说话人名称** | 点击说话人旁编辑按钮修改名称 |
| **只录音转文字（音频转文字）** | 仅录音记录可点击「转文字/音频转文字」触发非实时转写；受 §8.1 非实时转写次数上限约束 |

### 9.1 非实时转写次数上限

> 国内版**无转写时长/额度限制**（与海外订阅制不同），但对**同一条录音**的非实时转写（详情页-更多-音频转文字 / 重新转写）有**次数上限**：

| 操作次数 | 行为 |
|---------|------|
| 第 1 次 | 正常触发非实时转写 |
| 第 2 次 | 正常触发非实时转写 |
| **第 3 次及以后** | 详情页「更多」菜单中**不再展示「音频转文字」按钮**（入口隐藏）|

- 上限仅针对**非实时转写**；实时转写（录音过程中自动转）不受限
- 上限针对**单条录音**计数，不同录音独立计算
- ⚠️ 待确认：
  - 是否区分"音频转文字"（仅录音→转文字）与"重新转写"（已有转写→换语种重转）的计数
  - 重新转写次数用尽后，详情页对应按钮是否也隐藏或仅置灰
  - 次数是否跨设备同步

## 十、回听（音频播放）

| 功能 | 说明 |
|------|------|
| 点击转写文本播放 | 点击转写区某段文本，播放对应时段音频 |

## 十一、测试关注点

- 国内版仅有两种详情页类型（有转写/仅录音），无文档详情页
- 导出思维导图：详情页有 jpeg，批量导出无 jpeg
- 删除：已上云记录进回收站；云空间关闭时永久删除
- 无「移除」操作（国内版无 Space）
- 重新转写：覆盖 46 种转写语言（含方言/民族语言），优先级抽样按 P0-P4 分组（见 `reference-data/viaim-domestic/languages-readme.md`）
- **非实时转写次数**：同一录音最多 2 次，第 3 次时「更多」菜单中「音频转文字」按钮不展示（见 §9.1）
- 区分说话人：2~9 个或自动
- 查找与替换：替换当前 vs 全换
- 智能播放：是否正确跳过杂音/空白段
- 只录音记录点击「转文字」后流程
- **分享链接**：验证密码设置、有效期选项（永久/7天/30天/90天）、链接受限时的提示流程

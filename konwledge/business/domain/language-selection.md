# 语言选择弹窗规则

> 版本：V2.0.0 | 最后更新：2026-03
> 适用入口：设置页-录音设置 / 模式选择弹窗 / 录音页 / 详情页-音频转文字
> 同传（实时翻译与录音）的转写/翻译语种选择为独立体系，见 `features/viaim/recording/translate.md`
> 四个录音模式的转写语种列表和翻译语种列表**完全一致**，仅默认语种不同。
> 语种数据：转写 145 种（含地区变体）→ `reference-data/viaim/transcribe.csv`；翻译 80 种（无变体）→ `reference-data/viaim/translate.csv`

---

## 一、弹窗入口

| 入口 | 触发方式 | 说明 |
|------|---------|------|
| 设置页-录音设置 | 点击「转写语言」 | 关闭「录音实时转文字」开关后设置页依然显示转写语种 |
| 模式选择弹窗 | 点击「现场录音」下方语种区域 | 通话/音视频/现场共用转写语种 |
| 录音页（通话/音视频/现场） | 点击转写语言 | 录音中可切换 |
| 录音页（同传） | 点击转写/翻译语种 | 同传语种独立体系，见 translate.md |
| 详情页 | 音频转文字 → 点击选择语言 | 仅录音记录点击转文字后自动识别语种 |

---

## 二、转写语言选择弹窗结构

**两个 Tab：** 单语言 / 多语言

| 区域 | 内容 |
|------|------|
| 顶部 | Tab 切换（单语言 / 多语言）|
| 搜索框 | 输入关键词搜索语言 |
| 推荐语言区 | 最多 6 种，按最近使用排序 |
| 所有语言列表 | **145 种**语言（含地区变体），按 App 界面语言对应的排序规则排列 |
| 底部（多语言）| 确定按钮 |
| 关闭 | 右上角 X |

---

## 三、单语言模式

- 点击语言 → **直接确认**并关闭弹窗，进入转写
- 上次选择的语言在推荐区**第一位**且为选中状态
- 切换语言：点击新语言 → 弹窗关闭，转写语言立即切换
- 关闭弹窗（不选择语言）→ 保持原语言不变
- 选择后显示具体语言名称（如「意大利语」）

---

## 四、多语言模式

| 选中数量 | 确定按钮状态 | 行为 |
|---------|-----------|------|
| 0 种 | **置灰**不可点击 | — |
| 1 种 | **置灰**不可点击 | — |
| 2 种 | **高亮**可点击 | 点击确定 → 生效 |
| 3 种 | **高亮**可点击 | 点击确定 → 生效 |
| 第 4 种 | — | toast「选择的语言已达上限」，第 4 种不被选中 |

**交互细节：**
- 选中的语言按**选择顺序**显示在列表上方
- 取消选择：点击已选语言 / 点击其右侧 X 按钮
- 确认后记忆：下次打开弹窗自动选中上次确认的语言
- 未确认关闭弹窗：不保存本次选择，推荐语言不更新
- 各处显示「**多种语言**」而非具体语言名称（设置页/模式选择弹窗/录音页）

---

## 五、Tab 切换规则

| 操作 | 行为 |
|------|------|
| 单语言 → 多语言 | 多语言界面**无语言选中**，确定按钮置灰 |
| 多语言 → 单语言 | 单语言界面**无语言选中** |
| 多语言 → 单语言 → 多语言 | 多语言**保留**之前的选择状态 |
| 多语言选择但**未确认** → 切单语言 | 单语言显示之前的单语言选择（不受未确认的多语言影响）|
| 多语言已确认 → 再次打开 → 切单语言 | 单语言**无选中** |
| 搜索状态下切换 tab | **清空**搜索内容并退出搜索状态 |
| 频繁切换 | 无异常 |

## 六、推荐语言规则

### 转写推荐语言

**首次启动默认第一位（按 App 界面语言）：**

| App 界面语言 | 推荐第一位 |
|------------|----------|
| 中文 | 中文（普通话，简体）|
| 英语 | 英语（美国）|
| 日语 | 日语（日本）|
| 泰语 | 泰语（泰国）|
| 繁体中文 | 繁体中文（粤语）|
| 韩语 | 한국어（한국）|

**排序与数量：**
- 推荐语言最多 **6 种**
- 按**最近使用**顺序排列（最近用的排第一）
- 使用超过 6 种后仍只显示 6 种
- 选择但**未确认**不更新推荐列表
- 推荐语言可由**服务端配置**更新
- 转写语言、翻译语言的推荐语言及用户选择记录均**本地持久化**（不上云同步），换设备/重装App/清除缓存后重置为默认值

### 翻译推荐语言

**首次使用翻译时的默认推荐顺序（按 App 界面语言）：**

| App 界面语言 | 推荐顺序（6 种）|
|------------|--------------|
| 简体中文 | 中文、英语、日语、泰语、西班牙语、法语 |
| 繁体中文 | 繁体中文、英语、日语、简体中文、泰语、西班牙语 |
| 英语 | 英语、日语、简体中文、泰语、西班牙语、法语 |
| 泰语 | 泰语、英语、日语、简体中文、西班牙语、法语 |
| 日语 | 日语、英语、简体中文、泰语、西班牙语、法语 |
| 韩语 | 韩语、英语、日语、简体中文、泰语、西班牙语 |

- 上次使用的翻译语言在下次打开时排**第一位**

## 七、语言列表排序规则

| App 界面语言 | 所有语言列表排序方式 |
|------------|-----------------|
| 简体中文 | a-z 字母顺序 |
| 英语（美国）| a-z 字母顺序 |
| 繁体中文（粤语）| **笔画**顺序 |
| 日语 | **五十音**顺序 |
| 泰语 | **微软列表排序** |
| 西班牙语 | a-z 字母顺序 |
| 阿拉伯语 | **阿拉伯语字母**排序 |
| 韩语 | **微软列表排序** |
| 首次安装未拉取到语言 | 默认英文排序 |

- 转写语言总数：**145 种**（含变体）；翻译语言总数：**80 种**（无变体）

## 八、搜索功能

> 转写语言选择弹窗和翻译语言选择弹窗搜索规则一致。

| 规则 | 说明 |
|------|------|
| 触发方式 | 搜索框输入任意字符 → 立即进入搜索状态 |
| 搜索范围 | 按**语言名称**和**地区名称**匹配 |
| 大小写 | **不区分**大小写 |
| 关键词高亮 | 搜索结果中关键词**加粗**显示 |
| 实时刷新 | 边输入边更新搜索结果 |
| 空格支持 | 支持包含空格的搜索（前缀/后缀/中间空格均可搜索）|
| 字符长度 | 无上限限制 |
| 无匹配结果 | 显示空列表 |
| 清空按钮 | 清空搜索内容，显示全部语言 |
| 退出搜索 | 点击空白区域 / 键盘收起 / 清空按钮 → 退出搜索状态 |

**搜索状态下的选择行为：**

| 模式 | 搜索中选择语言 | 行为 |
|------|-------------|------|
| 单语言 | 点击搜索结果中的语言 | **直接确认**，关闭弹窗进入转写 |
| 多语言 | 点击搜索结果中的语言 | 语言选中显示在搜索框上方，**不退出搜索**，可继续搜索选择 |

---

## 九、翻译语言选择弹窗

> 翻译语言选择弹窗为独立弹窗（非转写语言选择弹窗的 tab）。

| 规则 | 说明 |
|------|------|
| 语言变体 | 翻译语言**不包含变体**（显示「英语」而非「英语（美国）」；显示「中文」而非「中文（普通话，简体）」）|
| 繁体中文 | 选择「繁体中文」时实际使用「繁体中文(台湾普通话)」翻译 |
| 语种范围 | **80 种**语言（无地区变体），包括当前正在转写的语种 |
| 搜索功能 | 与转写语言选择弹窗一致（见第八章）|
| 推荐语言 | 见第六章翻译推荐语言 |

## 十、语种同步规则

### 设置页 ↔ 模式选择弹窗 ↔ 录音中（三方同步）

以下三个入口的转写语言**互相同步**，任一处变更其余同步更新：
- 设置页 → 录音设置 → 转写语言
- 模式选择弹窗 → 现场录音下方语种
- 三个录音模式（通话/音视频/现场）录音中的转写语言

> 关闭「录音实时转文字」开关后，设置页依然显示转写语种，且同步关系不变。

### 开关关闭再重新开启

- 关闭「录音实时转文字」→ 重新开启后**沿用上次关闭时的语种**（单/多语言均保留）

### 非实时转写（详情页）的语种联动

| 「录音实时转文字」开关状态 | 详情页变更语种 | 设置/模式选择弹窗语种 |
|---------------------|-------------|----------------|
| **开启** | 详情页音频转文字/仅录音转文字中变更 | **不跟着变更**（独立）|
| **关闭** | 详情页变更后重新开启开关 | **跟着更新**为详情页变更后的语种 |

> 适用于所有切换组合：单→单、单→多、多→单、多→多。

### 详情页转文字的默认语种

| 记录类型 | 详情页转文字默认语种 |
|---------|----------------|
| 仅录音记录 | **无预设语种**，点击「转文字」后**自动识别语种**进行转写 |
| 实时转写记录 | 使用**该记录实时转写时选的语种** |

## 十一、多语言付费规则

> 多语言为付费功能，需在套餐基础上额外购买。套餐详情见 `domain/subscription.md`。

### Pro/Ultra 已付费多语言

- 正常使用多语言，**不弹升级弹窗**
- 打开语言选择弹窗点击多语言 tab → 正常进入

### 会员到期 + 之前已付费多语言

**触发规则：**

| 当前语种状态 | 用户操作 | 弹窗行为 |
|-----------|---------|---------|
| 多语言 | 开启录音 / 点击转写语言 | **同时弹出**语言选择弹窗 + 升级套餐弹窗（升级弹窗覆盖在上层）|
| 单语言 | 点击多语言 tab | 弹出升级套餐弹窗 |
| 此前用过多语言 | 录音中点击转文字 | **同时弹出**两个弹窗 |

**弹窗交互：**

| 操作 | 行为 |
|------|------|
| 关闭升级弹窗（X） | 显示语言选择弹窗（多语言 tab，保留之前的语种选择）|
| 语言选择弹窗中切换语言/取消已选中/点击确定 | **再次弹出**升级套餐弹窗 |
| 多语言 tab → 单语言 tab → 再切回多语言 tab | 弹出升级套餐弹窗 |
| 多语言 tab → 单语言 tab → 选择单语言 | **正常使用**（单语言不需要多语言付费）|
| 关闭语言选择弹窗（X） | 转写语言保持「多种语言」，界面无转写内容上屏 |
| 再次点击转写语言 | **再次同时弹出**两个弹窗 |
| 升级弹窗中付费成功 | 升级弹窗关闭 → 语言选择弹窗显示（保留之前选择）→ 可正常切换并确认 |

**无网络：**
- 不弹出语言选择弹窗和升级套餐弹窗
- 连接网络后弹出（录音中）；详情页连接网络后切换语言点确定提示无网络

**适用入口：** 设置页-录音设置 / 录音中 / 详情页（三处行为一致）

## 十二、测试关注点

**弹窗基础交互：**
- 单语言点击即确认；多语言 2~3 种高亮、0~1 种置灰、4 种 toast 上限
- Tab 切换：单→多无选中；多→单无选中；多→单→多保留多语言选择
- 未确认关闭不保存、不更新推荐语言

**搜索：**
- 实时搜索、不区分大小写、按语言+地区匹配、关键词加粗
- 单语言搜索选择直接确认；多语言搜索选择不退出搜索

**推荐语言：**
- 首次按 App 界面语言显示对应变体；之后按最近使用排序；最多 6 种
- 推荐语言及选择记录本地持久化，换设备/重装/清缓存后重置

**语种同步：**
- 设置页↔模式选择弹窗↔三个录音模式录音中 三方同步；关闭开关后设置页仍显示语种
- 实时转文字开启时：详情页变更不影响设置；关闭时：详情页变更影响设置
- 关闭再开启保留上次语种
- 仅录音记录无预设语种，点击转文字自动识别；实时转写记录沿用录制时语种

**翻译弹窗：**
- 翻译语种不含变体（"繁体中文"实际用"繁体中文(台湾普通话)"）
- 翻译列表包含所有语种

**多语言付费（会员到期）：**
- 多语言状态开启录音 → 同时弹两个弹窗；关闭升级弹窗后操作多语言 → 再弹升级弹窗
- 切到单语言 → 正常使用；付费成功 → 正常切换
- 无网络不弹弹窗，连网后弹

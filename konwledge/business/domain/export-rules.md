# 导出规则（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

## 一、导出入口与支持格式总览

**关键规则：不同入口支持的导出格式不同。**

| 入口 | 导出音频 | 导出原文件 | 导出转写 | 导出摘要 | 导出思维导图 |
|------|---------|----------|---------|---------|------------|
| 详情页「导出」| ✅ | ✅（文档格式）| txt / docx / pdf | txt / markdown / docx / pdf | **jpeg / markdown / pdf** |
| 记录列表批量操作 | ✅ | ❌ | txt / docx / pdf | txt / markdown / docx / pdf | markdown / pdf |
| Space 内批量操作 | ✅ | ✅（文档格式）| txt / docx / pdf | txt / markdown / docx / pdf | markdown / pdf |

**差异点：思维导图导出 jpeg 格式仅在详情页支持，批量导出（记录列表/Space）不支持 jpeg。**

---

## 二、详情页导出

### 导出音频
- 导出原始音频文件

### 导出原文件（文档类型）
- 导出格式：原文档格式后缀（如上传的是 PDF，导出也是 PDF）

### 导出转写

| 格式 | 支持 |
|------|------|
| TXT | ✅ |
| DOCX | ✅ |
| PDF | ✅ |

**转写导出选项：**
- 包含时间戳：开启 / 关闭
- 包含说话人名称：开启 / 关闭

### 导出摘要

| 格式 | 支持 |
|------|------|
| TXT | ✅ |
| Markdown | ✅ |
| DOCX | ✅ |
| PDF | ✅ |

### 导出思维导图（详情页）

| 格式 | 支持 |
|------|------|
| JPEG | ✅（**仅详情页**）|
| Markdown | ✅ |
| PDF | ✅ |

---

## 三、批量导出（记录列表 / Space 内）

### 批量导出数量限制

- 批量导出最多选择 **50 条**记录，超出时点击导出提示不可导出

### 导出音频
- 批量导出所选记录的音频；记录未下载音频时先下载再导出

### 导出转写

| 格式 | 支持 |
|------|------|
| TXT | ✅ |
| DOCX | ✅ |
| PDF | ✅ |

**转写导出选项：**
- 包含时间戳：开启 / 关闭
- 包含说话人名称：开启 / 关闭
- **每次打开格式弹窗不记忆上次选择及开关状态**

### 导出摘要

| 格式 | 支持 |
|------|------|
| TXT | ✅ |
| Markdown | ✅ |
| DOCX | ✅ |
| PDF | ✅ |

### 导出思维导图（批量）

| 格式 | 支持 |
|------|------|
| JPEG | ❌（批量不支持）|
| Markdown | ✅ |
| PDF | ✅ |

### Space 内专属：导出原文件
- 仅空间内上传的文档文件（pdf / docx / txt / pptx / markdown）可导出原文件
- 音频/录音类型快捷方式不可导出原文件
- 导出格式为原文档格式后缀

## 四、无内容时的处理规则

| 情况 | 行为 |
|------|------|
| 全部选中记录无对应内容（转写/摘要/思维导图）| 点击导出提示「无可导出内容」，终止流程 |
| 部分记录无内容 | 无内容记录跳过不生成文件；有内容记录正常导出 |
| 全部记录无音频 | 点击导出提示「无可导出内容」，终止流程 |
| 部分记录无音频 | 无音频记录跳过；有音频记录正常导出 |

---

## 五、文件命名规则

| 导出类型 | 压缩包命名 | 解压后文件命名 |
|----------|-----------|--------------|
| 音频 | `viaim(导出时间).zip`（如 `viaim(2025-12-31 10_57_20).zip`）| `记录名称_类型_创建时间.mp3`（中文环境类型=「音频」，英文=「Audio」）|
| 转写 | 同上 | `记录名称_Transcript_创建时间.{txt/docx/pdf}`|
| 摘要 | 同上 | `记录名称_Summary_创建时间.{txt/md/docx/pdf}`|
| 思维导图 | 同上 | `记录名称_Mind map_创建时间.{md/pdf}`|
| 原文件（Space）| 同上 | `记录名称_记录创建时间.原格式`（如 `会议记录_2025-12-31 14_05_17.pdf`）|

- 记录名称包含特殊字符（`/\:*?"<>|`）时自动替换为合法字符

## 六、导出进度弹窗规则

| 规则 | 说明 |
|------|------|
| 展示内容 | 导出进度 + 「请不要将应用放至后台」提示 + 停止导出按钮 |
| 停止导出 | 点击「停止导出」取消操作 |
| 导出成功 | 生成 zip 文件并调起系统分享面板 |
| 导出失败 | 网络异常时 toast 提示导出失败 |
| 切至后台 | 可能导致导出失败 |
| 分享取消 | 用户取消系统分享后留在当前页面并退出批量模式 |

---

## 七、翻译功能（详情页「更多」）

| 翻译类型 | 说明 |
|----------|------|
| 翻译转写 | 每段转写下展示翻译结果 |
| 翻译摘要 | 根据翻译语种重新生成摘要 |
| 翻译思维导图 | 根据翻译语种重新生成思维导图 |

---

## 八、测试关注点

- 详情页思维导图支持 jpeg，批量操作不支持（差异验证）
- 批量导出上限 50 条，超出时不可导出
- 转写导出「包含时间戳」「包含说话人名称」开关生效；每次不记忆上次状态
- 文件命名：中英文环境类型名称差异；特殊字符替换
- 导出压缩包格式正确，解压后文件可正常打开/播放
- 无内容时终止流程 vs 部分无内容时跳过的行为差异
- 导出过程中切后台可能失败
- 系统分享取消后退出批量模式
- 文档类型导出原文件格式后缀与上传格式一致（仅 Space 内）

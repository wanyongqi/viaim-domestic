# 删除与回收站规则（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

## 一、删除行为总览

| 操作来源 | 记录类型 | 删除行为 | 进回收站 |
|----------|----------|----------|---------|
| 记录列表左滑删除 | 已上云记录 | 删除到回收站 | ✅ |
| 记录列表左滑删除 | 未上云记录 | 直接永久删除 | ❌ |
| 记录列表批量删除 | 已上云记录 | 删除到回收站 | ✅ |
| 详情页「更多-删除」 | 音频原文件（从 record 进入）| 删除到回收站 | ✅ |
| 详情页「更多-移除」 | Space 内音频快捷方式（从 space 进入）| 移除到回收站 | ✅ |
| 详情页「更多-删除」 | Space 内文档原文件（从 space 进入）| 删除到回收站 | ✅ |
| Space 内左滑删除 | 文档原文件 | 删除到回收站 | ✅ |
| Space 内左滑移除 | 音频快捷方式 | 移除到回收站 | ✅ |
| Space 列表左滑删除 Space | 整个 Space | Space 内所有记录删除到回收站 | ✅ |
| 解析失败清除按钮 | 上传失败记录 | 直接永久删除 | ❌ |
| 回收站永久删除 | 任意 | 永久删除 | — |
| 回收站清空 | 所有 | 全部永久删除 | — |

## 二、回收站操作规则

### 可执行操作

| 操作 | 说明 |
|------|------|
| 恢复音频原文件 | 左滑快捷操作 |
| 恢复快捷方式 | 左滑快捷操作 |
| 恢复文档文件 | 左滑快捷操作 |
| 永久删除 | 左滑快捷操作 |
| 批量恢复 | 批量操作 |
| 批量永久删除 | 批量操作 |
| 清空回收站 | 清空全部 |

### 回收站内查看详情

| 能力 | 是否支持 |
|------|---------|
| 回听音频 | ✅ |
| 查看详情内容 | ✅（只读，不可操作）|
| 恢复 | ✅ |
| 永久删除 | ✅ |
| 编辑/操作 | ❌ |

---

## 三、保留时间

- 回收站内记录保留 **30 天**，30 天后自动永久删除。
- 在 30 天内可随时手动恢复或永久删除。

---

## 四、删除 vs 移除的区别

| 操作 | 对象 | 影响范围 |
|------|------|---------|
| **删除** | 原文件 | 原文件从记录列表消失，所有 Space 内的快捷方式随之置灰（不删除） |
| **移除** | Space 内快捷方式 | 只删除该 Space 内的快捷方式，原文件和其他 Space 内的快捷方式不受影响 |

---

## 五、移除确认弹窗规则（「同时删除原文件」选项）

Space 内移除音频快捷方式时，弹出「确认移除该来源吗？」二次确认弹窗，包含「同时删除记录中的原文件」勾选项：

| 选项状态 | 行为 |
|----------|------|
| **勾选（默认）** | 快捷方式 + 原文件同步删除到回收站；其他 Space 内快捷方式置灰 |
| **取消勾选** | 仅此 Space 快捷方式删除；原文件和其他 Space 快捷方式保留 |

- **每次打开弹窗均默认勾选，不记忆上次选择**
- 批量移除快捷方式时相同逻辑，勾选项同上

## 六、特殊规则汇总

- 未上云记录删除 → **直接永久删除，不进回收站**
- 解析失败记录清除 → **直接永久删除，不进回收站**
- 从 Space 删除整个 Space → 空间内所有记录（原文件 + 快捷方式）均删除到回收站；Record 列表中的源记录不受影响
- 删除空间时，空间内存在正在上传中的文件 → 不允许删除
- 删除空间时，音频文件处于解析中/解析失败 → 直接删除；记录列表和回收站不显示该解析中/失败状态
- 删除空间时，文档文件处于解析中/解析失败 → 直接删除，不进回收站
- 删除原文件后 → 所有 Space 内对应快捷方式置灰，点击提示「原文件已被删除」
- 从回收站恢复原文件 → 对应 Space 内置灰快捷方式恢复正常状态
- 置灰快捷方式左滑 → 仅显示「移除」，点击无二次确认直接删除，不进回收站

---

## 七、回收站恢复特殊场景

| 场景 | 行为 |
|------|------|
| 恢复文件，原所属 Space 仍存在 | 直接恢复到原 Space |
| 恢复文件，原所属 Space 已被删除 | 弹出 Space 选择弹窗，提示恢复到另一个 Space |
| Space 选择弹窗无其他 Space 可选 | 需先新建 Space 再恢复 |

> 添加到空间的弹窗展示、冲突处理、回收站恢复优先规则 → 详见 `domain/add-to-space-rules.md`

---

## 八、测试关注点

- 未上云 vs 已上云删除行为差异（前者不进回收站）
- 解析失败清除不进回收站
- 移除弹窗「同时删除原文件」勾选与取消的影响范围差异
- 每次打开移除弹窗均默认勾选（不记忆上次）
- 删除原文件后，Space 内快捷方式置灰而非删除
- 从回收站恢复后，置灰快捷方式是否恢复正常
- 删除 Space 时：上传中不允许删除；解析中/失败音频不进回收站；文档解析中/失败不进回收站
- 回收站恢复文件，原 Space 已删除时的 Space 选择弹窗
- 回收站详情页只读、不可操作
- 批量恢复 / 永久删除是否正常
- 清空回收站后是否彻底清除

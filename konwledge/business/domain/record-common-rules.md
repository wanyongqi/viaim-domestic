# 记录通用规则（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、记录来源与类型

| 来源 | 记录类型 | 筛选归类 | 创建时间定义 | 卡片右上角标识 | 展示位置 |
|------|----------|----------|-------------|--------------|---------|
| 连接耳机录音 | 通话录音 | 通话录音 | 录音开始时间 | 录音类型图标 | 记录列表 + Space |
| 连接耳机录音 | 现场录音 | 现场录音 | 录音开始时间 | 录音类型图标 | 记录列表 + Space |
| 连接耳机录音 | 音视频录音 | 音视频录音 | 录音开始时间 | 录音类型图标 | 记录列表 + Space |
| 连接耳机录音 | 实时翻译与录音 | 实时翻译与录音 | 录音开始时间 | 录音类型图标 | 记录列表 + Space |
| 闪录下载 | 通话闪录 | **通话录音**（自动归类）| 下载到本地时间 | 录音类型图标 | 记录列表 + Space |
| 闪录下载 | 现场闪录 | **现场录音**（自动归类）| 下载到本地时间 | 录音类型图标 | 记录列表 + Space |
| 上传音频 | 上传的音频 | 上传的录音 | 上传成功时间 | **文件格式标签**（如 MP3 / WAV）| 记录列表 + Space |
| 上传文档 | 上传的文档 | — | 上传成功时间 | 文件格式标签（如 PDF / DOCX）| **仅 Space 内** |

**说明：**
- 闪录下载后自动归类为对应录音类型（通话闪录 → 通话录音，现场闪录 → 现场录音）
- 上传文件（音频/文档）卡片右上角展示**文件格式标签**，而非录音类型图标
- 耳机录音与闪录下载卡片右上角展示**录音类型图标**
- 文档文件仅在 Space 内展示，记录列表不展示文档类型
- 记录可加入多个 Space，筛选「已加入空间」时只显示一次

## 二、记录状态与流程

### 2.1 状态类型

| 状态类型 | 适用来源 | 状态流程 |
|---------|---------|---------|
| **上云状态** | 耳机录音、闪录下载 | 等待上云 → 上云中（动画）→ 上云成功 / 上云失败（红色图标，可点击重试）|
| **解析状态** | 上传文件（音频/文档）| 解析中（紫色边框动画）→ 解析成功 / 解析失败（红色边框）|

### 2.2 各来源完整状态流程

**耳机录音：**

| 转写设置 | 状态流程 | 详情页内容 |
|---------|---------|-----------|
| 实时转写 | 录音结束 → 等待上云 → 上云中 → 上云成功 → 录音类型图标 | 有转写文字 + AI 生成内容 |
| 仅录音 | 录音结束 → 等待上云 → 上云中 → 上云成功 → 录音类型图标 | 仅音频播放 |

**闪录下载：**

| 转写设置 | 状态流程 | 详情页内容 |
|---------|---------|-----------|
| 开启自动转写 | 下载成功 → 等待上云 → 上云中 → 上云成功 → 录音类型图标 | 有转写文字 + AI 生成内容 |
| 未开启自动转写 | 下载成功 → 等待上云 → 上云中 → 上云成功 → 录音类型图标 | 仅音频播放 |

**上传音频文件：** 上传音频**必须转写，无法关闭转写**。

| 状态流程 | 详情页内容 |
|---------|-----------|
| 上传成功 → 解析中（紫色边框）→ 解析成功 / 解析失败 | 有转写文字 + AI 生成内容 |

**上传文档文件（仅 Space 内）：**

| 状态流程 | 详情页内容 |
|---------|-----------|
| 上传成功 → 解析中 → 解析成功 / 解析失败 | 摘要 / 预览 / 思维导图 / AI 对话 |

### 2.3 异常状态处理

**上云失败（耳机录音 / 闪录下载）：**
- 显示：红色云图标带 X
- 操作：点击可重新上传

**解析失败（上传文件）：**
- 显示：红色边框 + 解析失败文案 + 右上角清除按钮
- 操作：点击清除按钮直接永久删除（不进回收站）
- 限制：解析失败记录不可进入详情页 / 不可左滑 / 批量模式下**置灰不可选择，但保留卡片 UI**

## 三、卡片显示规则

> 以下规则适用于所有展示记录卡片的场景（Record 列表、Space 详情列表、记录库弹窗等）。

### 3.1 耳机录音卡片

**主标题优先级：**

| 条件 | 显示内容 |
|------|----------|
| 用户已手动修改标题 | 修改后的标题（不再生成智能标题） |
| 音视频录音获取到音频源标题 | 音频源标题 |
| 未修改标题且智能标题已生成 | 智能标题 |
| 未开启智能标题 / 无转写 | 录音开始时间（如"9月11,15:11"） |

**副标题优先级：**

| 条件 | 显示内容 |
|------|----------|
| 有转写且概览已生成 | 概览内容（替换转写文字） |
| 有转写且概览未生成 | 转写文字第一行（概览生成后自动替换） |
| 开启转写但无转写文本 | "转写无文本" |
| 通话录音仅录音（未开启转写）| "通话已录音" |
| 音视频录音仅录音 | "音视频已录音" |
| 现场录音仅录音 | "麦克风已录音" |

**右上角图标：**

| 录音类型 | 图标 |
|----------|------|
| 通话录音 | 通话录音图标 |
| 音视频录音 | 音视频录音图标 |
| 现场录音 | 麦克风图标 |
| 实时翻译与录音 | 翻译图标 |

**定位信息：** 有定位信息时显示（如"北京市海淀区"），超长省略；无定位或"未知"时不显示。

**上云状态展示【仅 Record 列表】：** 等待上云（图标）→ 上云中（动画）→ 上云失败（红色图标，可点击重试）→ 上云成功后不显示状态图标。

### 3.2 上传音频卡片

| 元素 | 显示规则 |
|------|----------|
| 主标题 | 文件名 + 格式后缀（如"会议录音.mp3"） |
| 副标题 | 解析中："解析中"；解析成功有转写："概览内容"；解析成功无转写："转写无文本"；解析失败："解析失败" |
| 右上角 | 格式标签（MP3/WAV/AAC/M4A/FLAC/WMA） |
| 时间 | **上传成功时间** |
| 时长 | 同耳机录音时长格式 |
| 解析成功 | 直接变为正常记录卡片，**不生成智能标题**，标题保持上传原始文件名 |

### 3.3 文档文件卡片【仅 Space 内展示】

| 元素 | 显示规则 |
|------|----------|
| 主标题 | 文件名 + 后缀（如"报告.pdf"） |
| 副标题 | 解析成功：概览内容（超长省略）；解析中/解析失败：对应文案 |
| 右上角图标 | 格式类型图标（PDF/DOCX/TXT/MD） |
| 时间 | 上传成功时间 |
| 文件大小 | B / KB / MB 格式，不显示小数点 |
| 解析失败卡片 | 红色边框 + 解析失败文案 + 清除按钮 |

### 3.4 快捷方式标识【仅 Space 内】

Space 内的音频快捷方式卡片，标题左侧额外展示**快捷方式图标**标识，以区分原文件。

### 3.5 省略规则

主标题/副标题超过一行省略显示（"..."）；特殊字符和 emoji 正常显示。

## 四、卡片标题生成规则

| 记录来源 | 标题生成逻辑 |
|----------|-------------|
| 耳机录音（通话/现场/音视频/实时翻译）| 开启智能标题：根据转写内容生成概括性标题；未开启或无转写：显示录音开始时间 |
| 音视频录音 | 优先使用音频源标题；未获取到则按智能标题逻辑处理 |
| 闪录下载 | 同耳机录音逻辑 |
| **上传音频** | **不生成智能标题，展示上传时的文件名** |
| **上传文档** | **不生成智能标题，展示上传时的文件名（含后缀，如「报告.pdf」）** |
| 用户手动修改 | 显示修改后的标题，不再生成智能标题 |

## 五、排序方式

| 排序选项 | 说明 |
|----------|------|
| 创建时间降序 | **默认**，排序按钮无选中高亮 |
| 创建时间升序 | 排序按钮选中高亮 |
| 修改时间降序 | 排序按钮选中高亮 |
| 修改时间升序 | 排序按钮选中高亮 |

**"创建时间"定义**：录音按开始时间、闪录按下载时间、上传文件按成功时间。

**排序记忆规则**：切换 Tab 后返回保持不变；重启 App / 退出重新登录恢复默认。Record 列表和 Space 各自独立记忆。

---

## 六、未读红点规则

| 规则 | 说明 |
|------|------|
| 本端生成录音/闪录且未查看 | 卡片标题左侧显示红点 |
| 点击进入详情页 | 红点消失，标记已读 |
| 其他端录音同步到本端 | 不显示红点（仅本端生成才显示） |
| 其他端查看 | 本端红点仍显示，需本端自行查看 |
| 所有记录已读 | 底部 Record Tab 红点消失 |

**适用范围**：Record 列表卡片、Space 详情列表卡片、底部 Tab 角标。

## 七、时间与时长通用格式

**时间格式：**

| 条件 | 格式 | 示例 |
|------|------|------|
| 今年 | MM-DD HH:mm:ss | 07-07 13:59:30 |
| 非今年 | YYYY-MM-DD HH:mm:ss | 2024-12-31 23:59:59 |

**时长格式：**

| 条件 | 格式 | 示例 |
|------|------|------|
| ≥ 1 小时 | Xh Xm Xs | 2h 11m 12s |
| ≥ 1 分钟 < 1 小时 | Xm Xs | 11m 12s |
| < 1 分钟 | Xs | 12s |

整数时显示"1h"或"30m"（无多余单位）。

## 八、测试关注点

- **记录类型标识差异**：耳机录音/闪录展示录音类型图标；上传文件展示格式标签
- **卡片标题**：上传音频/文档不生成智能标题，始终展示文件名；音视频录音优先取音频源标题
- **主标题优先级**：用户修改 > 音频源标题 > 智能标题 > 录音开始时间
- **副标题优先级**：概览 > 转写第一行 > 类型描述（通话已录音/麦克风已录音等）> 状态描述
- **"创建时间"定义差异**：录音=开始时间、闪录=下载时间、上传文件=成功时间
- **排序记忆**：切换 Tab 保持；重启/重登恢复默认；Record 和 Space 各自独立
- **未读红点**：仅本端生成的录音/闪录显示，其他端同步/查看不影响本端红点
- 闪录归类：通话闪录→通话录音、现场闪录→现场录音
- 文档类型不在记录列表展示，仅 Space 内可见
- 上云失败可重试；解析失败只能清除（永久删除，不进回收站）
- 解析失败记录不可左滑 / 批量模式下置灰不可选
- 快捷方式标识仅 Space 内展示

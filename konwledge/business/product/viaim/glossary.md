# viaim 术语表

| 术语 | 含义 |
|------|------|
| **记录** | 一条录音或上传文件在 App 中的一条数据；可来自耳机录音、闪录下载、上传音频、上传文档 |
| **记录列表** | 所有记录的列表入口；文档类型记录不在此展示，仅在空间内展示 |
| **空间 (Space)** | 原「分组」概念；可把记录加入空间，基于空间做 Vitana 问答等 |
| **上云** | 耳机录音/闪录的音频上传到云端；状态：等待上云 → 上云中 → 上云成功/上云失败 |
| **解析** | 上传文件（音频/文档）在服务端做转写或文本提取；状态：解析中 → 解析成功/解析失败 |
| **转写** | 语音转文字；可实时或上传后处理；与套餐转写时长、语种等限制相关 |
| **Vitana** | AI 助理；可将记录/空间内容作为背景知识进行提问；所有套餐均支持全部引用范围 |
| **闪录** | 耳机端独立触发录音，存于耳机，再通过 App 下载到手机 |
| **仅录音** | 不开启转写，仅保存音频；详情页仅音频播放，无转写与 AI 内容 |
| **双路通话录音** | 通话录音时我方与对方音频独立采集，音频播放页展示两条独立波形图，转写内容区分我方/对方两路；Pro3/Air2/Pro2 支持 |
| **单路通话录音** | 通话录音时我方与对方音频均会录制，但合为一路，音频波形图和转写内容不区分来源；Nano+/VIAIM Air 为单路 |
| **未激活会员** | V2.0.2 起新注册用户的默认状态，低于 Basic；需绑定设备或购买权益激活；不可添加记录/来源/上传，但可查看已有数据和 Vitana 对话 |
| **设备绑定** | V2.0.2 新增，搜索到未绑定设备后需先绑定才能使用；绑定设备可激活 Basic 会员 |
| **我的viaim设备** | V2.0.2 新增的设备管理页面；入口在我的→通用→录音设置上方；可查看/添加/解绑设备 |
| **安心录** | 网络通话安心录，开启后网络通话录音中来系统通话不中断；默认关闭；V2.0.2 新增首次使用引导弹窗流程 |
| **录音提示音说明页** | 欧美版达标固件耳机首次连接时弹出的全屏页面，用于设置录音开始时的语音提示音开关；关闭需勾选责任声明 |

更多状态与类型见 `domain/record-common-rules.md`。

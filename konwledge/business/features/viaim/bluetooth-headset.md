| 型号 | 别名/型号代码 | 支持录音类型 | 通话双路录音 | 闪录 | 噪声控制 | 佩戴检测 |
|------|-------------|------------|------------|------|---------|---------|

| **Air2** | opennote | 通话/音视频/现场/同传 | ✅ | ✅ | ❌ | ❌ |
| **Pro2** | recdot | 通话/音视频/现场/同传 | ✅ | ✅ | ✅ | ✅ |

| **VIAIM Air** | air | 通话/音视频/现场/同传 | ❌ 单路 | ❌ | ❌ | ❌ |

> **双路通话录音**：我方与对方音频独立采集，音频播放页展示两条独立波形图，转写内容区分我方/对方两路。

## 二、各耳机通用设置功能

以下功能所有型号均支持（除特殊说明外）：

| 功能 | 说明 |
|------|------|
| 右上角添加设备（V2.0.2新增）| 点击进入搜索设备页面；已连接时弹「请先断开」弹窗；搜索中按钮置灰；详见 `bluetooth-connect.md` 2.5 |
| 左右耳电量展示 | 展示当前左右耳电量 |
| 蓝牙连接管理 | 查看当前连接设备；断开连接；连接历史设备；连接新设备；删除配对关系 |
| 手势控制 | 双耳在线时点击弹出「是否更新耳机手势？」弹窗；详细规则与键值详见 `bluetooth-gesture.md` |
| 耳机音量 | 可调节耳机音量（Pro3/Pro2/Air2 支持）|
| 查找设备 | 通过播放声音查找耳机 |
| 固件升级 | 展示固件版本；有新固件时可 OTA 升级 |
| 关于设备 | 展示名称/型号/固件版本/序列号；可切换查看左耳/右耳/盒子信息 |
| 开盖显示电量（Android）| 开启：开盖时显示耳机电量；关闭：手动查看 |

### 蓝牙连接管理详细规则

**设备列表规则：**
- 按「最近成功连接时间」倒序排列（最近连接的排在最前）
- 最多同时保留 2 个连接设备；连接第 3 个设备时，断开最老且非发声的设备
- 同一时间只能有 1 个设备显示「发声中」标识；新设备抢占音频后标识随之转移
- 蓝牙连接管理页顶部展示「连接 Windows 设备使用注意事项」入口

**断开/删除设备操作规则：**

| 操作 | 设备状态 | 行为 |
|------|---------|------|
| 点击已连接设备 | 非发声状态 | 直接断开连接 |
| 点击已连接设备 | 发声状态 | 弹出二次确认断开弹窗；确认后断连 |
| 删除（移除配对）已连接设备 | — | 弹出移除耳机配对二次确认弹窗；点击取消保持连接；点击确认后断连并删除 |
| 删除未连接设备 | — | 弹出移除配对弹窗；确认后删除 |
| 点击未连接设备 | — | 尝试重新连接；连接失败弹出连接失败弹窗 |
| 删除全部配对设备 | — | 回到设备详情页，显示未连接状态 |

### 查找设备规则

- 可分别播放左耳/右耳，或双耳同时播放（同时发出嗡鸣声）
- 点击播放按钮（无论耳机是否佩戴）→ 弹出**音量警告弹窗**
- 弹窗点击「播放」→ 对应耳机发出嗡鸣声
- 点击关闭/取消/弹窗外 → 弹窗关闭，不播放

### 固件升级详细规则

**有新固件时的入口：**
- 首页左上角显示固件升级提示（点击跳转固件更新页）
- 耳机设置页图片下方显示固件升级提示栏（点击跳转）；「固件更新」列表项显示红点
- 闪录时长不足时优先显示**闪录不足提示**（不显示固件升级提示），但固件更新红点仍保留

**升级前提条件：**

| 条件 | 行为 |
|------|------|
| 单/双耳未入盒 | 弹出「请将双耳放入充电盒」提示弹窗；点击关闭/好的/弹窗外关闭 |
| 任一组件（左耳/右耳/盒子）电量 < 30% | 弹出「请连接电源」提示弹窗；点击关闭/好的/弹窗外关闭 |
| 已入盒 + 电量充足 | 可直接开始升级 |
| 已入盒 + 电量不足 + 已插电源 | 可开始升级 |

**升级中断情况（均导致升级失败）：**
- 手机锁屏或返回系统首页 → 升级失败
- 杀掉 App → 升级失败；再次打开 App 连接耳机正常
- 耳机关盖 → 升级失败，显示升级错误页；可点击「重试」（若未入盒则先弹入盒提示）或「以后再说」回到固件更新页

**升级后状态：**
- 升级成功：显示成功页（新固件版本 + 「知道了」按钮）→ 返回固件更新页显示当前版本；首页/设置页提示消失；红点消失
- 升级失败：首页提示/设置页提示/红点均保留

### 关于设备详细规则

- 展示：耳机名称、型号、固件版本、序列号
- 点击「型号」「固件版本」「序列号」任意字段 → 可切换查看左耳/右耳/盒子的对应信息

### 开盖显示耳机电量规则（Android）

| 条件 | 行为 |
|------|------|
| 未授权系统悬浮窗权限 | 点击开关 → 跳转系统悬浮窗权限页；授权后返回 App，开关已打开 |
| 已授权悬浮窗权限 | 点击开关直接打开 |
| 开关已开启 → 耳机关盖再开盖重连 | 弹出耳机电量弹窗 |
| 关闭开关后 → 耳机关盖再开盖重连 | 不弹出电量弹窗 |

### 耳机音量规则（Pro3/Air2/Pro2）

**iOS：**
- App 音量滑块调节 ↔ 系统音量始终同步（双向联动）

**Android（系统已开启绝对音量）：**
- App 音量滑块 ↔ 系统媒体音量同步

**Android（系统已关闭绝对音量）：**
- App 音量滑块与系统音量独立调节，互不影响

**其他规则：**
- 可调节到 0%（无声）和 100%（最大音量）
- 耳机关盖断连后重新连接：音量滑块恢复为**上次调整的值**

### 3.6 佩戴检测

| 型号 | 支持 |
|------|------|
| Pro3 | ✅ 自动播放/暂停 / 自动接听来电 / 自动通透模式 / 自动通话降噪模式 |
| Air2 | ❌ |
| Pro2 | ✅ 同 Pro3 |

**各功能默认值（耳机重置后）：**

| 功能 | 默认状态 |
|------|---------|
| 自动播放/暂停 | **开启** |
| 自动来电接听 | 关闭 |

**触发逻辑：**
- 勾选「自动播放/暂停」：取下耳机自动暂停；重新佩戴自动播放

- 勾选「自动通透模式」+ 当前为降噪模式：只佩戴单耳时自动切换到通透模式

### 3.7 耳塞检测（贴合度测试）

| 型号 | 支持 |
|------|------|
| Pro3 | ✅ |
| Air2 | ❌ |
| Pro2 | ✅ |
| Nano+ | ✅ |
| VIAIM Air | ❌ |

**耳塞检测流程：**
1. 进入「耳塞检测」页面（路径：耳机设置 → 耳塞检测）
2. 点击「开始测试」：
   - 单/双耳未出盒 → 弹出「请双耳佩戴」提示弹窗
   - 双耳已佩戴 → 进入佩戴贴合度检测中（loading 状态）
3. 检测中：可点击左上角「取消」→ 回到开始测试页
4. 检测完成后展示结果：
   - 双耳贴合度良好：展示良好状态（与 UI 一致）
   - 单/双耳贴合度不佳：展示不佳状态；可点击「调整角度后再次测试」
   - 右上角显示「完成」按钮 → 点击回到耳机设置页
   - 底部显示「再次测试」按钮 → 点击重新检测

## 四、测试关注点

**型号与功能差异：**
- 各耳机型号设置页功能差异（按型号逐一对照上表验证）
- VIAIM Air：不支持闪录、不支持噪声控制、不支持声音风格、耳机设置中无闪录入口
- Nano+ / VIAIM Air：名称不可修改
- 音质偏好「声音质量优先」与双设备连接/音视频录音的互斥逻辑（仅 Pro3/Air2/Pro2）
- 佩戴检测：Air2/VIAIM Air 不支持
- 双设备连接：Nano+/VIAIM Air 默认支持无开关；Pro3/Air2/Pro2 有开关控制

**手势控制：** 详见 `bluetooth-gesture.md`

**录音开启播报提示音：**
- 仅欧美版 Pro2/Pro3/Air2 且固件达标才显示该设置项
- 单耳连接时无法切换开关状态（toast 提示，状态不变）
- 关闭需二次确认弹窗
- 开关状态跨设备（同一耳机双连）自动同步
- **V2.0.2 首次连接说明页**：欧美版达标固件首次连接弹出全屏说明页（开关/声明/确认交互），详见 bluetooth-connect.md

**蓝牙连接管理：**
- 第 3 个设备连接时，断开最老且非发声设备（非当前发声设备）
- 点击发声中设备需二次确认弹窗；点击非发声设备直接断开
- 删除当前连接设备：二次确认弹窗；删除非当前连接设备：移除配对弹窗（无二次确认）

**查找设备：**
- 无论耳机是否佩戴，点击播放均弹出音量警告弹窗

**噪声控制：**
- Nano+ 仅三档主模式，无子模式（验证设置页不出现程度滑块/子选项）
- Pro2/Pro3 切换主模式后子模式设置保留

**声音风格：**
- 各型号风格列表不同：Pro3（6项）、Air2（4项）、Pro2（18项）、Nano+（22项）
- 自定义音效：可进入设置页调节；已选自定义可修改

**佩戴检测：**
- 耳机重置后：自动播放/暂停默认开启；其余三项默认关闭

**耳机音量：**
- iOS：App 音量与系统音量双向同步
- Android 绝对音量开/关 → 同步/独立调节行为不同
- 断连重连后显示上次音量值

**固件：**
- 固件升级：未升级 vs 已最新 两种状态展示
- 升级前提：双耳入盒 + 电量 ≥ 30%（否则弹提示弹窗）
- 升级中：锁屏/退出/杀进程/关盖 均会导致失败
- 闪录不足时优先显示闪录提示（固件升级提示被遮盖，但红点保留）

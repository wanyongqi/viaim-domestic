# 账号与设置功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、账号设置

| 功能项 | 说明 |
|--------|------|
| 名字 | 支持设置/修改名字，不超过 10 字，不可全空格 |
| 绑定邮箱（无邮箱时）| 进入绑定邮箱流程 |
| 更换邮箱（有邮箱时）| 点击弹出确认弹窗，继续后进入更换流程 |
| 解绑手机号 | 点击手机号弹出二次确认弹窗可解绑（需有其他登录方式）|
| 设置密码（无密码）| 进入设置密码流程（仅 Apple 账号需先绑定邮箱）|
| 修改密码（有密码）| 进入修改密码流程 |
| 绑定 Google | 拉起 Google 授权弹窗绑定 |
| 解绑 Google | 弹出二次确认弹窗，满足条件可解绑 |
| 绑定 Apple | 拉起 Apple 授权弹窗绑定（iOS 可主动绑定；Android 仅显示已绑定状态）|
| 解绑 Apple | 弹出二次确认弹窗，满足条件可解绑 |
| 接受个性化邮件开关 | 开启：可收到邮件；关闭：不收邮件（**有邮箱时才显示**）|
| 注册方式 | 展示注册账号时使用的方式（Email / Phone / Google / Apple）|
| 退出登录 | 点击即退出，无需确认 |
| 删除账号 | 勾选协议 → 身份验证 → 二次确认 → 删除成功 |

> 登录注册详细流程、图形验证码规则、密码规则详见 `features/viaim/login-register.md`

---

### 1.1 绑定邮箱流程

```
点击绑定邮箱 → 图形验证码 → 输入新邮箱 → 图形验证码 → 新邮箱验证码
→ [无密码账号] → 设置密码页 → 绑定成功
→ [已有密码账号] → 直接绑定成功
```

| 规则 | 说明 |
|------|------|
| 输入已占用邮箱 | 提示「该邮箱已被使用」|
| 输入非法格式邮箱 | 输入框变红，下一步置灰 |
| 仅绑定 Apple（无手机号）| 新邮箱验证页无「使用手机号验证」入口 |
| 绑定后 | 可用邮箱+密码登录，数据保持一致 |

### 1.2 更换邮箱流程

```
点击邮箱地址栏 → 弹出确认弹窗（显示当前邮箱掩码）→ 点击「继续更换」
→ 图形验证码 → 旧邮箱验证（已绑定手机号时底部可切换手机号验证）→ 输入新邮箱
→ 图形验证码 → 新邮箱验证码 → 更换成功
```

| 规则 | 说明 |
|------|------|
| **已绑定手机号** | 旧邮箱验证页底部显示「使用手机号进行身份验证」入口，可切换 |
| **未绑定手机号** | **无手机号验证入口** |
| 输入旧邮箱或已占用邮箱 | 提示「邮箱已注册」|
| 更换成功后 | 可用新邮箱登录，数据保持一致 |

### 1.3 解绑手机号规则

| 场景 | 行为 |
|------|------|
| 有邮箱+密码（或有 Google/Apple）| 解绑成功，手机号栏隐藏 |
| 仅手机号一种登录方式（无邮箱+密码，无三方）| **解绑失败**，提示「解绑失败，请先设置其他登录方式」|
| 解绑成功后用原手机号登录 | 提示「请先添加其他登录方式」|

### 1.4 设置密码流程

```
点击登录密码（未设置）
→ [仅 Apple 且无邮箱] → 提示「请先绑定邮箱」（终止）
→ [有邮箱] → 图形验证码 → 邮箱验证码 → 设置新密码页 → 设置成功
→ [有手机号且无邮箱] → 图形验证码 → 手机验证码 → 设置新密码页 → 设置成功
```

### 1.5 修改密码流程

```
点击登录密码（已设置）→ 图形验证码 → 邮箱验证（已绑定手机号时可切换手机号验证）→ 输入新密码页 → 修改成功
```

| 规则 | 说明 |
|------|------|
| 同时绑定邮箱+手机号 | 优先进入邮箱验证，底部可切换手机号验证 |
| 仅绑定邮箱 | 只有邮箱验证，无切换入口 |
| 仅绑定手机号 | 只有手机号验证，无切换入口 |
| 修改成功后 | 邮箱+新密码、手机号+新密码均可登录 |

### 1.6 绑定/解绑 Google

**绑定：**
- 拉起 Google 授权弹窗 → 授权成功且未被其他账号占用 → 绑定成功
- 该 Google 账号已绑定其他用户 → 失败，提示「该账号已绑定至其他用户，请更换账号」

**解绑：**
| 场景 | 行为 |
|------|------|
| 仅 Google 一种登录方式（无密码/无邮箱/无手机）| **解绑失败**，提示「解绑失败，请先设置其他登录方式」|
| 有邮箱但无密码 | **解绑失败**，提示「解绑失败，请先设置登录密码」|
| 有邮箱+密码 / 有手机号 | 解绑成功，注册方式仍显示 Google |

### 1.7 绑定/解绑 Apple

**绑定：**
- iOS：可主动绑定；Android：不显示绑定入口，但可显示已绑定状态并解绑
- 该 Apple 账号已绑定其他用户 → 失败，提示「该账号已绑定至其他用户，请更换账号」

**解绑：**
| 场景 | 行为 |
|------|------|
| 仅 Apple 一种登录方式（无密码/无邮箱/无手机）| **解绑失败**，提示「解绑失败，请先设置其他登录方式」|
| 有邮箱但无密码 | **解绑失败**，提示「解绑失败，请先设置登录密码」|
| 有邮箱+密码 / 有手机号 | 解绑成功 |

### 1.8 删除账号流程

```
进入删除账号页 → 勾选「我已阅读并同意以上内容」（未勾选按钮置灰）
→ 点击「确认删除账号」→ 身份验证（按登录方式不同）→ 二次确认弹窗 → 确认 → 删除成功
```

**身份验证方式（按优先级）：**

| 账号登录方式 | 验证方式 |
|-------------|---------|
| 有邮箱（无论是否有手机）| 图形验证码 → 邮箱验证码（**有手机号时**底部显示「切换手机号验证」入口）|
| 仅手机号 | 图形验证码 → 短信验证码 |
| 仅 Google | 拉起 Google 授权验证 |
| 仅 Apple | 拉起 Apple 系统级验证（Face ID / Touch ID）|
| Google/Apple + 邮箱 | 图形验证码 → 邮箱验证 |
| Google/Apple + 手机号（无邮箱）| 图形验证码 → 手机验证 |

**删除成功后：**
- 跳转至未登录首页（Splash Screen），toast 提示「账号已删除」
- 其他设备同时登录时**强制下线**

### 1.9 注册方式显示规则

| 注册方式 | 显示值 |
|---------|--------|
| 邮箱注册 | Email |
| 手机号注册（老版本）| Phone |
| Google 注册 | Google |
| Apple 注册 | Apple |

## 测试关注点

**账号设置：**
- 修改密码/更换邮箱/解绑手机等操作均需图形验证码（人机校验）
- 邮箱/手机号切换验证入口：**仅当账号同时绑定了邮箱和手机号时才显示切换入口**
- 解绑/删除前需校验是否还有其他可用登录方式
- 仅 Apple 账号设置密码需先绑定邮箱
- 删除账号后其他设备被强制下线
- Google 解绑后注册方式仍显示 Google
- Android 不显示「Apple 未绑定」入口

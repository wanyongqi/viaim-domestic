## 一、录音入口与模式选择弹窗

**录音按钮入口：**
- 已连接耳机：底部 Tab 栏显示录音按钮，动态效果与 UI 一致
- 未连接耳机：底部 Tab 栏**不显示**录音按钮

**模式选择弹窗：**
- 点击录音按钮 → 弹出模式选择弹窗，展示 4 种录音类型
- 关闭方式：点击右上角 X / 弹窗区域外
- 点击任意录音类型 → 关闭弹窗，进入对应录音页

**模式弹窗中转写语言显示规则：**

> 弹窗内转写语言仅在两处展示，通话录音和音视频录音下方均**不单独显示**语种。

| 展示位置 | 显示内容 | 显示条件 |
|--------|--------|--------|
| **「现场录音」下方** | 通话/音视频/现场三种录音**共用**的转写语种 | 设置页**已开启**实时录音转文字时显示；未开启则**不显示** |
| **「实时翻译与录音」下方** | 当前源语言 + 目标语言 | 始终显示 |

**语言选择弹窗焦点规则（实时翻译与录音）：**
- 点击**源语言**侧 → 弹出转写语言选择弹窗，列表定位到当前选中语言
- 点击**目标语言**侧 → 弹出翻译语言选择弹窗，列表定位到当前选中语言
- 现场录音转写语言 与 实时翻译语种**互不影响**

**Android 高品质音频（声音质量优先）与音视频录音冲突规则：**
> 适用耳机：Pro3 / Pro2 / Air2

- 已开启高品质音频 → 点击音视频录音 → 弹出**「无法使用音视频录音」**弹窗
- 弹窗内点击「关闭声音质量优先」→ 弹出关闭确认弹窗
  - 点击**关闭**：重启耳机关闭高品质音频 → 之后可正常开启音视频录音
  - 点击**取消 / X / 弹窗外**：关闭弹窗，维持原状
- 弹出无法使用弹窗后点击**取消 / X / 弹窗外**：直接关闭弹窗

## 二、录音类型与触发条件

| 录音类型 | 触发条件 | 说明 |
|----------|----------|------|
| **通话录音** | 当前连接耳机且存在通话状态（网络通话/系统通话）| 双路：Pro3/Air2/Pro2；单路：Nano+/Air |
| **音视频录音** | 当前正在播放音视频 | 录制音视频音频内容 |
| **现场录音** | 连接耳机，录制现场音频 | 耳机采集音频 |
| **实时翻译与录音** | 手机麦克风录音，实时转写+翻译+TTS 播报 | 同传听译模式 |

## 五、通用功能概览

以下功能在多种录音中共享（具体适用范围见矩阵）：

| 功能 | 说明 |
|------|------|
| **修改标题** | 点击标题弹窗；上限 **50 字符**（可输入超过 50 字符但确认按钮**置灰**无法保存）；X 清空输入框；取消/X/弹窗外关闭不保存 |
| **实时转写** | 可选单个语种或 2-3 个多语种转写 |
| **实时翻译** | 开启实时翻译（外翻外）；关闭则只转写不翻译 |
| **文字直播** | 复制链接 / 生成二维码 / 更多；**四种录音均支持**；详见 text-live.md |
| **功能栏** | 默认展开；可收起/展开；左右滑动查看全部按钮 |
| **结束录音** | 点击结束按钮 / 系统返回手势（弹二次确认弹窗）/ 耳机键值（Pro2/Air2/Pro3）|
| **来电挂断继续** | 录音中来电接听/挂断后可继续录音；通话录音绑定安心录详见 call.md |

**智能标题生成规则：**
- 录音中**未修改标题** → 结束后自动生成智能标题（仅实时转写模式，仅录音模式不生成）
- 录音中**已修改标题** → 结束后不覆盖，保留手动标题

## 八、测试关注点

**录音入口与模式弹窗：**
- 录音按钮仅连接耳机时显示
- 模式弹窗语种：「现场录音」下方显示三种录音共用转写语种（受开关控制），「实时翻译与录音」下方显示源+目标语言
- Android Pro2/Pro3/Air2：高品质音频开启时无法开启音视频录音

**通用功能：**
- 标题统一上限 50 字符，超出按钮置灰
- 系统返回手势：4 种录音均弹二次确认弹窗
- 智能标题：实时录音未修改标题 → 生成；仅录音/已修改标题 → 不生成

**各模式关注：**
- **通话录音**：安心录绑定来电规则、双路/单路、佩戴提示、蓝牙干扰中断 → 详见 call.md
- **音视频录音**：被动暂停（音源停止）、获取源标题 → 详见 common-rules.md
- **现场录音**：无独有功能，规则与通用一致
- **实时翻译与录音**：麦克风收音、转写时长为 0 无法进入、播报/速度/语种记忆 → 详见 translate.md

**记录保存：** → 详见 common-rules.md
- 杀 App 时：仅录音 → 保存；实时转写 10s 内 → 不保存
- **实时翻译与录音杀 App**：无转写内容 → 无论时长均不保存

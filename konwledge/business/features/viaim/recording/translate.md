# 实时翻译与录音专属功能

> 以下规则**仅适用于实时翻译与录音**。
> 实时翻译与录音无「仅录音」模式、无「区分说话人」、无「标记」功能。
> 使用**手机麦克风**收音（非耳机麦克风），实时转写 + 翻译 + TTS 播报。
> 通用规则见 [common-rules.md](common-rules.md)，功能矩阵见 [README.md](README.md)。

---

## 一、UI 结构

| 区域 | 内容 |
|------|------|
| 顶部 | 录音时长计时器 + 标题（可编辑）|
| 中部上半 | 转写原文区域（默认 50%）|
| 中部下半 | 译文区域（默认 50%）|
| 功能栏 | 转写语言、翻译语言、文字直播、播报开关、朗读速度等 |
| 底部 | 暂停、结束录音、收起/展开功能栏 |

**区域显示规则：**
- 原文/译文各占 50%，**每次新开启录音重置为 50/50**
- 原文和译文**独立滚动**，互不影响
- 可通过拖拽中间手柄调整比例（拖拽超过 72pt 切换大屏/小屏窗口；不足 72pt 弹回原位）
- 空状态提示：原文区「此处将显示原文」、译文区「此处将显示译文」（随 App 界面语言本地化显示）
- 有内容上屏后提示文案消失
- 内容不在当前屏幕时，展示「滚动到下方」按钮；点击后双区域同步滚动至最新位置
- 持续语音输入时新内容自动跟随滚动，不展示「滚动到下方」按钮

## 二、首次启动默认语种规则

| App 界面语言 | 转写默认语言 | 翻译默认语言 |
|------------|-----------|-----------|
| 中文 | 英语 | 中文 |
| 日语 | 英语 | 日语 |
| 泰语 | 英语 | 泰语 |
| 繁体中文 | 英语 | 繁体中文 |
| 阿拉伯语 | 英语 | 阿拉伯语 |
| **英语** | **西班牙语** | **英语** |
| 韩语 | 英语 | 韩语 |
| 其他语言 | 英语 | 该界面语言 |

> 翻译默认语言 = App 当前界面语言；转写默认语言 = 英语（仅 App 语言为英语时转写默认为西班牙语，避免与翻译语言重复）。

## 三、语种切换与记忆

- 录音中可点击转写/翻译语种处切换语言
- 点击任一侧弹出语言选择弹窗，弹出时有**一次 light 震动反馈**
- 弹窗焦点默认落在**点击的一侧**，列表定位到当前选中语言
- 手动更改语种后**记忆**：下次开启同传录音继续使用上次设置
- 设置页选择的转写语言 与 同传选择的语言**互相独立**

---

## 四、播报开关规则

| 状态/操作 | 行为 |
|----------|------|
| 默认状态 | **开启**（译文通过耳机 TTS 播报 + 译文高亮）|
| 关闭播报 | toast「语音播报已关闭」；耳机停止播报，译文高亮消失 |
| 开启播报 | toast「语音播报已开启」；从**新生成的译文**开始播报并高亮 |
| 暂停录音时 | 播报开关状态保持不变 |
| 暂停时关闭播报 → 继续录音 | 新译文不播报 |
| 暂停时未播完的译文（开关为开）| 暂停后剩余译文继续通过耳机播报完成 |
| 结束录音 → 再次开启新录音 | 播报开关**重置为开启** |
| 多次快速切换 | 均可正常切换，无异常 |

---

## 五、朗读速度

- 可选档位：更慢 / 慢 / 正常 / 快 / 更快
- 调整后**立即生效**（新合成的译文按新速度播报）
- 速度设置弹窗不会自动关闭
- 速度设置**记忆并同步云端**：下次开启同传仍为上次设置

---

## 六、暂停时的播报行为

- 暂停录音：停止收音，**不打断**进行中的翻译/合成/播报，继续完成
- 暂停状态下耳机正常将未播报部分播报完毕
- 继续录音后：新转写**不打断**之前未完成的翻译/合成/播报，**按顺序**执行
- 暂停时间不计入转写时长

---

## 七、转写时长为 0 时的行为

| 用户等级 | 操作 | 行为 |
|---------|------|------|
| Basic/Pro（转写时长为 0）| 点击「实时翻译与录音」| **无法进入**，弹出转写时长不足弹窗（含升级权益按钮）|
| Basic（时长为 0）| 弹窗点击升级 | 默认选中 Pro；升级后正常使用 |
| Pro（时长为 0）| 弹窗点击升级 | 默认选中 Ultra；升级后正常使用 |
| Ultra | 点击「实时翻译与录音」| 正常进入 |

> 与其他三种录音不同：通话/音视频/现场录音转写时长为 0 时仍可进入仅录音模式；实时翻译与录音**直接拦截不允许进入**。

**录音中转写时长耗尽：**
- 原文区域提示转写额度已耗尽，可继续仅录音，无播报
- 功能栏功能可正常点击，无异常
- 切换转写语言/引擎重连后仍无法转写，可继续仅录音

---

## 八、设置页「实时录音转文字」开关与同传的关系

- 设置页关闭「实时录音转文字」→ 开启实时翻译与录音时弹 toast：**「实时翻译与录音将消耗转写额度」**
- 同传录音**不受**设置页「实时录音转文字」开关影响，正常转写翻译
- 设置页选择的默认语种 与 同传语种互相独立

---

## 九、麦克风权限

| 场景 | 行为 |
|------|------|
| 首次使用 | 弹出 App 麦克风授权弹窗 → 点击「去开启权限」→ 弹出系统授权弹窗 |
| 系统授权 → 允许 | 进入录音页 |
| 系统授权 → 不允许 | 返回 App 主界面 |
| 授权弹窗 → 点击区域外/下拉 | 关闭弹窗，返回上级页面 |
| 拒绝后再次点击同传 | 跳转到系统权限设置页面，需手动授权 |
| 覆盖安装 App（已授权）| **不需要**重新授权 |
| 卸载重装 App | **需要**重新授权 |

## 四、我的viaim设备与解绑（V2.0.2 新增）

> 入口：我的 → 通用 → 录音设置上方「我的viaim设备」  
> 绑定流程见 `features/viaim/bluetooth-connect.md` 五、设备绑定流程

### 4.1 入口与可见性

- 所有用户类型（未激活/Basic/Pro/Ultra）均可见
- 点击进入设备管理页

### 4.2 设备管理页

**空状态：** 占位图 + 暂无设备 + 添加设备按钮

**有设备状态：** 设备列表 + 底部全宽添加设备按钮

**设备卡片展示：** 设备图标 + 蓝牙名称 + 左耳SN + 解绑红色按钮

**当前已连接设备：** 名称旁显示绿色圆点且置顶展示

**设备数量：** 一个账户绑定多台设备，**无数量上限**

### 4.3 添加设备操作

| 当前状态 | 操作 | 行为 |
|---------|------|------|
| 空状态 | 点击添加设备 | 进入搜索连接流程 |
| 有设备 + 当前无连接 | 点击添加设备 | 进入搜索连接流程 |
| 有设备 + 当前有连接 | 点击添加设备 | 弹「请先断开当前设备」弹窗 |

- 请先断开弹窗：知道了 / X / 弹窗外关闭

### 4.4 解绑

**入口：** 设备管理页 → 点击已绑定设备的解绑按钮 → 弹出解绑确认弹窗

**弹窗内容：** 设备图片 + 文案 + 红色警告 + 继续解绑按钮 + 取消按钮

**关闭方式：** 取消 / X / 弹窗外 / 按住顶部下拉 → 关闭弹窗不执行解绑

**解绑行为（按连接状态）：**

| 设备状态 | 继续解绑后行为 |
|---------|-------------|
| 设备已连接 | 清除闪录 + 断开连接 + 解除绑定 |
| 设备未连接 | 仅解除绑定（闪录数据保留在耳机）|

**闪录数据清理规则（按绑定历史）：**

| 场景 | 闪录处理 |
|------|---------|
| 全新设备（从未绑定过）→ 首次绑定 | **不清除**闪录数据 |
| 曾绑定过的设备 → 同一账户重新绑定 | **清除**闪录数据 |
| 曾绑定过的设备 → 不同账户绑定 | **清除**闪录数据 |
| 解绑期间耳机独立产生新闪录 → 重新绑定 | 同样**清除** |
| 解绑后已下载到App的闪录数据 | **不受影响** |

**解绑后状态：**

| 项目 | 解绑后状态 |
|------|----------|
| 权益 | 保持不变（不因解绑降级）|
| 设备管理列表 | 设备移除 |
| 重新连接 | 仍按MAC定向搜索被解绑设备 |
| 被解绑设备不在附近 | 定向搜索超时 → 通过添加设备连接新设备B → MAC更新为B |

### 4.5 缓存同步

- 绑定/解绑成功后 → 本地缓存同步更新

## 五、录音设置

> 部分设置项带有「上云」「本地」「不上云」标注，说明该设置影响的处理链路。

| 设置项 | 显示条件 | 说明 |
|--------|---------|------|
| 闪录设置入口 | **连接耳机时显示；连接 VIAIM Air 耳机时不显示** | 点击进入闪录设置页 |
| 录音实时转写（上云）| 始终显示 | 开启：录音模式选择弹窗中现场录音下方展示转写语言；开启通话/音视频/现场录音后实时转写文本。关闭：录音模式选择弹窗不展示转写语言；只录音不转写 |
| 闪录同步后自动转写（上云）| **连接 VIAIM Air 耳机时不显示** | 开启：闪录下载后自动转写文字；关闭：闪录下载后不自动转写 |
| 转写语言（本地）| 始终显示 | 可选单语言或多语言；选择后开启通话/音视频/现场录音，转写语言按选择执行 |
| 自动区分说话人（上云）| 始终显示 | 开启：开启录音实时转写时，启动录音后自动按说话人区分转写文本；关闭：不区分说话人 |
| 位置信息记录（上云）| 始终显示 | 开启：录音记录展示位置信息；关闭：位置信息为空 |
| 允许在桌面查看录音状态 | **Android 专属** | 开启：实时录音时可悬浮窗查看录音 |
| 自动开始录音（不上云）| 始终显示 | 开启：耳机收到通话信号时打开 App 自动开启通话录音；关闭：有通话时打开 App 不自动开启 |
| **网络通话安心录**（V2.0.2新增）| 始终显示 | 最底部展示；默认关闭；开启后进入通话录音安心录自动激活；详见 `recording/call.md` |

### 5.1 Flash Record 触发行为规则

> 仅在连接支持闪录的耳机时，录音设置中才显示「Flash Record 闪录设置」入口；**连接 VIAIM Air 耳机时不显示**。

**所有支持闪录耳机（Pro3/Pro2/Air2/Nano+）— 设置选项：通话-自动选择 / 通话-闪存模式 / 现场-闪存模式**

| 设置选项 | 有通话 + 已连接App | 有通话 + 未连接App | 无通话 + 已连接App | 无通话 + 未连接App |
|--------|-----------------|-----------------|-----------------|-----------------|
| 通话-自动选择 | 启动 **App 端通话录音** | 启动耳机本地通话闪录 | — | — |
| 通话-闪存模式 | 启动耳机本地通话闪录 | 启动耳机本地通话闪录 | — | — |
| 现场-闪存模式 | — | — | 启动耳机本地现场闪录 | 启动耳机本地现场闪录 |

## 六、AI 设置

| 设置项 | 说明 |
|--------|------|
| 职业身份 | 输入框最多 **50 个字符**，类型不限，可正常修改保存；行业信息收集填写会**覆盖**手动填写内容 |
| 默认 AI 模型 | 设置同步至：个性化模版、模版库、重新生成摘要/待办/脑图 |
| 聊天机器人模型 | 设置同步至：Vitana |
| 输出语言 | 选择后，智能标题、概览、摘要待办、脑图、Vitana 回复均按所选语言输出；切换转写语言或 App 界面语言**不影响**已设置的 AI 输出语言 |
| 摘要模版个性化开关 | 开启：显示自定义模板入口，录音结束后按自定义模板生成摘要待办；关闭：不显示自定义模板入口，按默认模板生成 |
| 自定义模板 | 展示自定义模版列表，支持创建/编辑/删除/查看/应用 |
| 智能标题 | 开启：显示「包含日期」开关，根据内容生成概括性标题；关闭：不生成智能标题；**仅对录音类型生效，上传音频/文档不生成智能标题，展示上传时的文件名** |
| 智能标题包含日期 | 开启：标题含日期前缀（如"10-17 录音标题"）；关闭：纯概括性标题，无日期；**关闭智能标题开关后再重新开启，该子开关保持原状态不重置** |

### 6.1 AI 大模型分级与套餐限制

| 模型类型 | 模型名称 | 可用套餐 |
|--------|---------|---------|
| 免费模型 | Gemini 3 Flash | Basic / Pro / Ultra |
| 免费模型 | GPT-5 Mini | Basic / Pro / Ultra |
| 免费模型 | Claude Haiku 4.5 | Basic / Pro / Ultra |
| 付费模型 | GPT-5.2 | Pro / Ultra（Basic 不可选，显示皇冠图标）|
| 付费模型 | Gemini 3.1 Pro | Pro / Ultra |
| 付费模型 | Claude Sonnet 4.5 | Pro / Ultra |

**默认模型规则：**
- Basic 用户：默认 AI 模型和聊天机器人模型均为 **GPT-5 Mini**
- Pro/Ultra 用户：默认 AI 模型和聊天机器人模型均为 **GPT-5.2**
- Basic 升级到 Pro/Ultra 后：两个模型**自动切换为 GPT-5.2**

**AI 输出语言默认值规则：**
- 新用户 / 未设置：默认与 App 界面语言一致
- 修改 App 界面语言：**不影响**已设置的 AI 输出语言
- 切换转写语言：**不影响**已设置的 AI 输出语言

### 6.2 摘要模版详细规则

**开关与入口：**
- 「摘要默认模板开关」新用户默认**关闭**
- 开关开启后显示「自定义模板」设置项；关闭后隐藏
- 录音结束后：开关开启 + 已选模板 → 按自定义模板生成；开关关闭 / 无自定义模板 → 按默认格式生成
- 「摘要默认模板开关」关闭时，详情页重新生成摘要弹窗中**仍可选择自定义模板**

**字符上限：**
- 模板名称：最多 **20 个字符**，超出无法输入
- 提示词：最多 **2000 个字符**（V2.0.2 从500提升至2000），超出保存时 Toast 提示

**创建模板流程：**
1. 进入模板库弹窗 → 点击「创建」→ 弹出创建模板窗口
2. 填写模板名称（必填，否则保存按钮置灰）；提示词可选填
3. 提示词支持手动填写自然语言描述，也可直接填写结构化提示词
4. 保存成功 → 返回模板库弹窗，**新建模板自动选中**

**模板库弹窗操作：**
- 支持左右滑动浏览模板列表；首/末模板边界不轮询
- 点击模板卡片 → 弹出预览弹窗（显示模板名称和提示词内容）
- 预览弹窗中左右滑动可切换其他模板（无法轮询）
- 选中模板后**不可取消选中**（只能切换到其他模板）
- AI 设置页：上次选中的模板**下次打开自动选中**
- 详情页模板库：**每次打开均为未选中状态**

**编辑/删除模板：**
- 编辑：带入现有名称和提示词；不保存直接关闭则不修改
- 删除：弹出二次确认弹窗 → 确认删除后**自动选中下一个模板**；若无其他模板则展示空模板库弹窗
- 删除确认弹窗点击关闭/取消：返回预览弹窗

**多端同步：**
- 创建/编辑/删除操作**多端实时同步**

**套餐限制：**
- Basic：最多 1 个；Pro：最多 10 个；Ultra：最多 20 个
- 超限后点击「创建」→ 弹出「达到上限」弹窗 → 引导升级

## 十、测试关注点

**我的viaim设备与解绑（V2.0.2）：**
- 所有用户类型均可见入口，设备数量无上限
- 添加设备按钮行为随连接状态变化（无连接→搜索，有连接→弹窗拦截）
- 已连接解绑：清除闪录+断连+解绑（三步）；未连接解绑：仅解绑（闪录留在耳机）
- 闪录清理依据绑定历史：首次绑定不清除、重新绑定清除（同账户/不同账户均清除）
- 解绑后权益不变、已下载闪录不受影响
- 解绑后MAC定向搜索仍指向被解绑设备

**录音设置：**
- 闪录设置和「闪录同步后自动转写」在连接 VIAIM Air 耳机时不显示
- 「允许在桌面查看录音状态」仅 Android 显示，需悬浮窗权限
- 自动开始录音开关**不跨设备同步**；其他录音设置开关均上云同步
- Flash Record「自动选择」模式：已连 App → App 端录音；未连 App → 耳机本地闪录
- Android 12 及以上：连接耳机不再弹出定位权限请求
- 智能标题仅对录音类型生效，不影响上传文件的文件名展示

**AI 设置：**
- 默认 AI 模型影响范围：个性化模版/模版库/重新生成，**不**影响 Vitana（Vitana 单独有聊天机器人模型设置）
- 输出语言影响智能标题/概览/摘要待办/脑图/Vitana 回复语言；切换 App 界面语言或转写语言均不影响
- Basic 可选3个免费模型（Gemini 3 Flash / GPT-5 Mini / Claude Haiku 4.5）；Pro/Ultra 可选全部6个模型；付费模型显示皇冠图标
- Basic 升级后两个模型自动变 GPT-5.2（原 GPT-5 Mini 设置被覆盖）
- 自定义模板受套餐数量限制（Basic 1个、Pro 10个、Ultra 20个）；多端实时同步
- 摘要模板名称上限 **20 字符**；提示词 **2000 字符**（V2.0.2 从500提升至2000）
- **安心录开关（V2.0.2）**：录音设置最底部，默认关闭；开启后进入通话录音安心录自动激活
- AI 设置页打开模板库自动选中上次选中项；详情页每次打开均未选中

**权限管理：**
- iOS 无悬浮窗权限项；Android 有定位/悬浮窗两项
- 「允许在桌面查看录音状态」依赖悬浮窗权限，未授权时开启引导去授权

**技术支持：**
- 帮助中心跳转目标按当前连接耳机型号不同（未连接跳总页）
- 复制日志后自动跳转在线客服页（Toast 1秒后跳转）

**隐私政策多语言：**
- 日语界面 → 日文版；其他所有界面语言（含韩语）→ 英文版

**主题切换：**
- 深色/浅色/跟随系统三种模式切换后界面整体跟随变化

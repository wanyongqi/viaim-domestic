# Vitana AI 助理功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、概念

Vitana 是 viaim 的 AI 助理，可以以记录或 Space 内容为背景知识进行问答。

---

## 二、入口

| 入口 | 默认引用范围 |
|------|------------|
| 底部 Vitana Tab | 全部记录 |
| Space 内 | 当前 Space |
| 通话/音视频/现场/实时翻译录音详情页 | 当前记录 |
| 上传音频详情页 | 当前记录 |
| 上传文档详情页 | 当前记录 |

---

## 三、引用范围

引用范围与套餐无关，**所有套餐均支持全部引用范围**：

| 引用范围 | 说明 |
|----------|------|
| 选择某一记录 | 根据单个记录内容进行问答 |
| 选择全部 | 根据全部记录范围进行问答 |
| 选择某一空间 | 根据空间内所有记录内容进行问答 |

---

## 四、支持的大模型

> 模型列表详见 `subscription.md` 第五节。当前支持免费模型（GPT-5 Mini / Gemini 3 Flash / Claude Haiku 4.5）和付费模型（GPT-5.2 / Gemini 3.1 Pro / Claude Sonnet 4.5，需 Pro/Ultra）。

**默认模型**：Basic 默认 GPT-5 Mini；Pro/Ultra 默认 GPT-5.2。
**Basic 选收费模型**：直接弹出升级权益弹窗（不是生成后弹出）。
**降级行为**：收费模型自动切回 GPT-5 Mini；免费模型保持不变。

> 详细 AI 模型权限规则见 `subscription.md` 第五节。

---

## 五、Vitana 对话功能

| 功能 | 说明 |
|------|------|
| 输入方式 | 文字输入 / 语音输入 |
| 新对话 | 点击「新对话」按钮创建新会话 |
| 历史对话 | 点击「历史对话」进入历史列表 |
| 历史对话继续 | 点击某一历史对话，可继续发问 |
| 删除历史对话 | 点击删除可删除某个历史对话 |
| 重新回答 | 最新回复可点击「重新回答」重新生成 |
| 反馈 | 最新回复可「喜欢/不喜欢」反馈（点击或长按）|
| 打断回复 | 回复中，输入新问题点击发送，打断当前回复，回复新问题 |

---

## 六、测试关注点

- 引用范围与套餐无关，所有套餐均支持「单条/全部/某一空间」
- 付费大模型 Basic 用户**点击即弹升级弹窗**（非生成后弹出）
- **AI 默认模型**：Basic = GPT-5 Mini；Pro/Ultra = GPT-5.2；降级时收费模型自动切回免费
- 历史对话管理（新建/继续/删除）
- 打断回复后新问题是否正确回复
- 节点验证：日本节点、新加坡节点

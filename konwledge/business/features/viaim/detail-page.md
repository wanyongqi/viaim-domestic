# 详情页面功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、详情页类型与内容差异

| 详情页类型 | 适用来源 | 主要内容 |
|-----------|---------|---------|
| **有转写的音频详情页** | 耳机录音（开启转写）/ 上传音频（**必须转写**）/ 闪录下载（开启自动转写）| 音频播放 + 转写 + 摘要&待办 + 思维导图 + AI 对话（Vitana）|
| **仅录音详情页** | 耳机录音（关闭转写）/ 闪录下载（未开启自动转写）| 仅音频播放，无转写和 AI 内容 |
| **文档详情页** | 上传文档 | 预览（原文）+ 摘要&待办 + 思维导图 + AI 对话（Vitana），**无音频播放** |

---

## 二、标题与概览

| 功能 | 说明 |
|------|------|
| **智能标题** | 开启智能标题开关后，根据转写内容自动生成标题；适用于所有录音类型（通话/现场/音视频/实时翻译），**上传音频和文档不生成智能标题，展示上传时的文件名** |
| **修改标题** | 点击标题旁修改按钮弹出弹窗，可修改录音标题 |
| **概览** | 根据转写内容自动生成概览（精简摘要）|

---

## 三、导出功能

详见 `domain/export-rules.md`，详情页导出支持：

| 导出类型 | 格式 |
|----------|------|
| 导出音频 | 原音频文件 |
| 导出原文件（文档）| 原文档格式 |
| 导出转写 | txt / docx / pdf（含时间戳/说话人选项）|
| 导出摘要 | txt / markdown / docx / pdf |
| 导出思维导图 | **jpeg / markdown / pdf**（详情页独有 jpeg）|

---

## 四、翻译功能（更多菜单）

| 翻译类型 | 说明 |
|----------|------|
| 翻译转写 | 每段转写下展示翻译结果 |
| 翻译摘要 | 根据翻译语种重新生成摘要 |
| 翻译思维导图 | 根据翻译语种重新生成思维导图 |

---

## 五、删除/移除（更多菜单）

| 入口 | 操作 | 行为 |
|------|------|------|
| 从 record 进入详情页 | 删除 | 删除音频原文件到回收站 |
| 从 space 进入音频快捷方式详情页 | 移除 | 仅移除该 Space 内快捷方式，原文件不受影响 |
| 从 space 进入文档详情页 | 删除 | 删除文档原文件到回收站 |

---

## 六、摘要&待办 Tab

| 功能 | 说明 |
|------|------|
| **模板库** | 选择模板重新生成摘要；支持创建/编辑/删除/查看/应用自定义模版 |
| **预制模版** | 见 `reference-data/viaim/预制模版.csv` |
| **自定义模板数量上限** | Basic:1 / Pro:10 / Ultra:20 |
| **复制** | 仅复制文本 / 复制文本与格式 |
| **重新生成** | 填写问题或预期后点击重新生成 |
| **待办勾选** | 可勾选/取消勾选复选框，标记待办完成状态 |
| **反馈** | 喜欢 / 不喜欢 |

---

## 七、思维导图 Tab

| 功能 | 说明 |
|------|------|
| 复制脑图 | ✅ |
| 重新生成 | 点击可重新生成脑图 |
| 全屏 | 脑图横屏展示 |
| 放大/缩小 | 加号放大，减号缩小，还原按钮自适应位置 |
| 节点交互 | 点击节点可展开/收起；双指可放大/缩小 |
| 全部展开/收起 | 展开或收起全部节点 |

---

## 八、转写 Tab

| 功能 | 说明 |
|------|------|
| **编辑** | 点击编辑按钮进入编辑文本页，可编辑转写原文/译文 |
| **显示译文** | 开启/关闭译文显示 |
| **复制记录** | 仅复制文本 / 复制原文和翻译 |
| **重新转写** | 可选单个或 2-3 个转写语种重新转写 |
| **区分说话人** | 开启：选择说话人数（2~9/自动），按声纹分说话人；关闭：不区分 |
| **查找与替换** | 输入关键词查找高亮，可替换当前或全部关键词 |
| **修改说话人名称** | 点击说话人旁编辑按钮修改名称 |
| **只录音转文字** | 仅录音记录可点击「转文字」触发转写 |

---

## 九、预览 Tab（文档详情页专属）

| 功能 | 说明 |
|------|------|
| 预览原文 | 可查看解析成功的原文档内容 |

---

## 十、回听（音频播放）

| 功能 | 说明 |
|------|------|
| 点击转写文本播放 | 点击转写区某段文本，播放对应时段音频 |
| 音频条播放/暂停 | 点击音频条播放；点击暂停 |
| **智能播放** | 开启后自动掠过杂音和空白 |
| **播放速度** | 可选择播放倍速（0.5x / 1x / 1.5x / 2x 等）|

---

## 十一、测试关注点

- 三种详情页类型的内容差异（有转写/仅录音/文档）
- 文档详情页无音频播放器
- 导出思维导图：详情页有 jpeg，批量导出无 jpeg
- 从 record vs space 进入：删除 vs 移除行为差异
- 自定义模板套餐上限（Basic:1 / Pro:10 / Ultra:20）
- 重新转写：单语种 vs 多语种（2-3 个）
- 区分说话人：2~9 个或自动
- 查找与替换：替换当前 vs 全换
- 智能播放：是否正确跳过杂音/空白段
- 只录音记录点击「转文字」后流程
- 节点验证：日本节点 / 新加坡节点

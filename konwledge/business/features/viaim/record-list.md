# 记录列表功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、展示范围

仅展示**音频类型**记录（耳机录音、闪录下载、上传音频）。文档类型记录不在记录列表展示，仅在 Space 内展示。

> 卡片显示规则、排序方式、未读红点、时间与时长格式 → 详见 `domain/record-common-rules.md`

---

## 二、筛选方式【Record 列表专属】

| 筛选维度 | 可选项 |
|----------|--------|
| 录音类型 | 全部（默认）/ 通话录音 / 音视频录音 / 现场录音 / 实时翻译与录音 / 上传的录音 |
| 音频位置 | 已加入空间 / 未加入空间 |

**筛选规则：**
- 筛选只能单选（选新项自动取消原选项）
- 通话录音筛选包含通话闪录下载的记录；现场录音筛选包含现场闪录下载的记录
- 上传的录音不含闪录下载的记录
- 同一记录加入多个空间 → 筛选"已加入空间"只显示一次
- 筛选结果为空 → 显示空状态，筛选图标保持高亮，不显示上传/录音按钮
- 选择非默认筛选项 → 筛选图标选中高亮；选回"全部" → 恢复默认颜色
- 筛选记忆规则同排序：切换 Tab 保持；重启 App / 重登恢复默认

## 三、左滑快捷操作【Record 版本】

| 操作 | 条件 | 行为 |
|------|------|------|
| 添加到空间 | 已上云且有转写的记录 | 可选择单个或多个空间添加，在 Space 中创建快捷方式 |
| 添加到空间 | 仅录音记录（无转写） | **不可添加**（即使已上云） |
| 添加到空间 | 未上云记录 | **不可添加** |
| 删除 | 已上云记录 | 删除到回收站 |
| 删除 | 未上云记录 | 直接永久删除，不进回收站 |

> 添加到空间的弹窗展示与冲突处理规则 → 详见 `domain/add-to-space-rules.md`
> 删除行为详细规则 → 详见 `domain/delete-recycle-rules.md`

## 四、批量操作【Record 版本：3 按钮】

### 进入与退出

| 操作 | 说明 |
|------|------|
| 进入 | 长按任意记录卡片 |
| 退出 | 点击退出按钮 / Android 系统返回手势 / 关闭冲突弹窗后自动退出 |
| 解析中/解析失败记录 | **置灰不可选择，但保留卡片 UI**（与 Space 批量模式一致） |
| 底部按钮 | 「导出」+「删除」+「添加到」三个按钮 |
| 未选择 | 底部按钮置灰不可点击 |

### 导出类型（4 种）

| 操作 | 说明 |
|------|------|
| 导出音频 | 导出所选记录的音频文件 |
| 导出转写 | txt / docx / pdf（含时间戳/说话人选项） |
| 导出摘要 | txt / markdown / docx / pdf |
| 导出思维导图 | markdown / pdf（**无 jpeg**，与 Space 批量一致） |

> **与 Space 批量导出的差异**：Record 无"导出原文件"（Space 多此项，用于文档原文件导出）。
> 导出格式、命名、进度弹窗等详细规则 → 详见 `domain/export-rules.md`

### 批量删除

选择记录后点击删除 → 已上云记录删除到回收站。

> 删除行为详细规则 → 详见 `domain/delete-recycle-rules.md`

### 批量添加到空间【Record 专属】

选择记录后点击添加到 → 可批量添加到单个或多个空间。仅录音记录（无转写）和未上云记录不可添加。

> 添加冲突处理规则 → 详见 `domain/add-to-space-rules.md`

## 五、上传入口【Record 列表专属】

### 入口位置

| 列表状态 | 入口位置 |
|---------|---------|
| 有记录 | 右上角**添加记录按钮** |
| 无记录（空状态） | 中部**添加记录按钮** |

### 未激活会员限制（V2.0.2 设备绑定需求）

未激活会员（未绑定设备的新用户）点击添加记录按钮 → **弹出解锁 Basic 会员弹窗**，引导用户绑定 viaim 设备激活 Basic 权益。

> 解锁弹窗内容及设备绑定完整规则 → 详见 `features/viaim/device-binding.md`

### 上传功能

记录列表添加记录入口 → 仅支持**音频**上传（不支持文档）。

选择音频文件后弹出配置弹窗：

| 配置项 | 说明 |
|--------|------|
| 区分说话人 | 开启则根据选择说话人数区分说话人 |
| 添加到空间 | 可选择单个或多个空间，上传成功后在对应 Space 创建快捷方式 |
| 转写消耗提示 | Basic/Pro 显示"将消耗 XX 分钟 / 剩余 XX 分钟"；Ultra 不显示 |

> 详细上传规则（格式/大小/队列/校验/容量）→ 详见 `domain/upload-rules.md`

## 六、测试关注点

- Record 列表仅展示音频类型，文档不出现
- **筛选**：通话录音筛选包含通话闪录；现场录音包含现场闪录；上传的录音不含闪录
- 同一记录加入多个空间 → 筛选"已加入空间"只显示一次
- 筛选结果为空时显示空状态，筛选图标保持高亮
- **左滑**：已上云→回收站；未上云→永久删除；**仅录音（无转写）即使已上云也不可添加到空间**
- **批量 3 按钮**：导出/删除/添加到（Space 批量只有 2 按钮，无"添加到"）
- 批量导出 4 种（无"导出原文件"，Space 多此项）
- 批量模式下解析中/解析失败记录**置灰可见**（与 Space 一致）
- **上传入口**：仅音频，不支持文档；弹窗含区分说话人/添加到空间/转写消耗提示
- 节点验证：日本节点、新加坡节点

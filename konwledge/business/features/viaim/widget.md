# 小组件功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03  
> 覆盖：Android 悬浮窗、Android 通知栏、iOS 负一屏小组件、iOS 灵动岛

---

## 一、Android — 桌面悬浮窗

### 前提条件
需开启悬浮窗权限（可在「我的-权限管理」中授权）。

### 状态与内容

| 状态 | 显示内容 |
|------|---------|
| 未连接耳机 | 不显示悬浮窗 |
| 连接耳机，未开启录音 | 显示悬浮窗；点击 X 可收起；显示三种录音按钮，点击可开启对应录音 |
| 通话录音中 | 左侧展示时间/录音类型/正在录音图标（只录音）或转写及翻译文字（实时转写，双路仅显示对方）|
| 音视频录音中 | 左侧展示时间/录音类型/正在录音图标（只录音）或转写及翻译文字（实时转写）|
| 现场录音中 | 同音视频录音 |
| 同传听译中 | 左侧展示转写及翻译文字 |
| 通话录音闪录中 | 左侧展示时间、左/右耳正在存储录音 |
| 现场录音闪录中 | 左侧展示时间、左/右耳正在存储录音 |

### 录音中悬浮窗功能

| 功能 | 说明 |
|------|------|
| 转文字语种选择 | 可选择转写语种 |
| 调整字号 | 点击「a」调小，点击「A」调大（仅对悬浮窗生效）|
| 标记（录音中）| 点击标记图标，当前前后 10s 转写文本标红 |
| 悬浮窗尺寸 | 可调整悬浮窗大小 |
| 静音（通话录音）| 点击静音按钮可静音麦克风，再次点击解除 |
| 结束录音 | 点击结束图标结束录音 |

## 二、Android — 通知栏

### 前提条件
需开启通知栏权限。

### 状态与内容

| 状态 | 显示内容 |
|------|---------|
| 未连接耳机 | 不显示通知栏 |
| 连接耳机，未开启录音 | 显示三种录音按钮 + 降噪切换 + 左右耳电量；可切换降噪（通透/关闭/降噪）；点击录音可开启对应类型 |
| 通话录音中 | 显示时间/录音中；点击可进入录音页面；可切换降噪 |
| 音视频录音中 | 同通话录音中 |
| 现场录音中 | 同通话录音中 |
| 同传听译中 | 同通话录音中 |
| 通话录音闪录中 | 显示三种录音（点击不可开启录音）+ 降噪切换 + 左右耳电量 |
| 现场录音闪录中 | 同通话录音闪录中 |

## 三、iOS — 负一屏小组件

### 使用方式
在 iOS 负一屏中添加 viaim 小组件。

### 状态与内容

| 状态 | 显示内容 |
|------|---------|
| 未连接耳机 | 提示「手机蓝牙未连接耳机」|
| 连接耳机，未开启录音 | 显示三种录音按钮 + 降噪切换 + 左右耳电量；可切换降噪；点击可开启对应录音 |
| 通话录音中 | 显示录音中；点击可进入录音页面；可切换降噪 |
| 音视频录音中 | 同通话录音中 |
| 现场录音中 | 同通话录音中 |
| 同传听译中 | 同通话录音中 |
| 通话录音闪录中 | 显示三种录音（点击不可开启录音）+ 降噪切换 + 左右耳电量 |
| 现场录音闪录中 | 同通话录音闪录中 |

## 四、iOS — 灵动岛

### 适用设备
iPhone 14 Pro 及以上支持灵动岛的机型。

### 触发条件
录音中退出 App 时顶部展示灵动岛。

### 灵动岛模式

| 模式 | 触发条件 | 展示内容 |
|------|---------|---------|
| 紧凑式 | 录音中，不在 App 页面 | 图标 + 时间 |
| 扩展式 | 长按灵动岛区域 | 时间 + 结束按钮 + 耳机图标 + 耳机电量 |
| 极简式 | viaim 与其他 App 同时使用灵动岛 | 仅展示图标 |
| 锁屏式 | 录音中锁屏 | 时间 + 耳机图标 + 耳机电量 |

### 支持的录音类型

| 录音类型 | 退出 App 后是否展示灵动岛 |
|----------|------------------------|
| 通话录音中 | ✅ |
| 音视频录音中 | ✅ |
| 现场录音中 | ✅ |
| 同传听译中 | ✅ |
| 未录音 | ❌ 不显示 |

## 六、测试关注点

- **Android 悬浮窗**：未连接耳机时不显示；闪录中点击不可开启新录音
- **iOS 灵动岛**：紧凑式/扩展式/极简式/锁屏式四种模式切换
- **iOS 灵动岛**：仅 iPhone 14 Pro+ 支持，普通机型不显示（仅顶部提示）
- **通知栏/负一屏**：闪录中三种录音按钮不可点击开启新录音
- 悬浮窗字号调整仅对悬浮窗本身生效，不影响 App 主界面
- 降噪切换：通透/关闭/降噪三档，在通知栏/负一屏均支持
- 从悬浮窗/通知栏点击录音区域可进入 App 对应录音页面

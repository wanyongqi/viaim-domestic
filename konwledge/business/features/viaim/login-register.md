# 登录注册功能规格（viaim）

> 版本：V2.0.0 | 最后更新：2026-03

---

## 一、闪屏页与用户协议

| 场景 | 行为 |
|------|------|
| 首次安装打开 App | 弹出「重要提醒」弹窗，包含隐私政策和服务条款链接 |

| 元素 | 说明 |
|------|------|
| 登录按钮 | 进入邮箱登录页 |

## 三、图形验证码规则（人机校验）

图形验证码为**数学计算题**，需输入正确答案后才能继续。

| 规则项 | 说明 |
|--------|------|
| 触发场景 | 邮箱注册、邮箱/手机号找回密码、账号设置中的绑定邮箱/更换邮箱/解绑手机/修改密码/删除账号 |
| 有效期 | 图片 **5 分钟**后过期，过期后点击确定提示「图片已过期」|
| 刷新限制 | 1 分钟内最多刷新 **20 次**，超出提示「验证码获取频繁，请稍后再试」|
| 输入为空 | 确定按钮置灰，不可点击 |
| 输入错误 | 提示「验证码错误」，可重新输入；多次错误也可重新计算，无锁定 |
| 输入字符（非数字）| 提示「答案错误，请重新输入」|
| 关闭弹窗 | 点击关闭不发送验证码 |
| 验证通过 | 发送验证码，弹窗关闭，开始倒计时 |

---

## 四、密码规则

所有涉及密码设置/修改/重置的场景均适用：

| 规则 | 要求 |
|------|------|
| 最小长度 | 至少 **8 位** |
| 字母要求 | **同时包含大写和小写字母** |
| 数字要求 | **至少包含 1 位数字** |
| 实时校验 | 输入过程中三条规则逐一点亮 |
| 确认密码校验 | 实时对比，不一致时持续显示 error 状态「两次输入的密码不一致」|
| 确认一致后 | 输入框恢复正常，下一步按钮点亮 |

## 五、邮箱登录流程

```
邮箱输入页 → [已注册] → 密码输入页 → 登录成功
                ↓ [未注册]
             弹出「去注册」弹窗（可跳转注册页，邮箱自动填充）
```

### 邮箱输入规则
- 不允许输入空格、中文
- 最大 **254 个字符**
- 实时格式校验：输入一半时显示 warning，输入完成且合法时点亮下一步
- 密码本自动填充：空格/中文自动过滤；格式不正确则输入框变红
- 返回上一页：输入框清空

### 密码输入页
- 自动聚焦输入框，键盘拉起
- 支持明文/密文切换
- 输入错误密码：输入框抖动变红，toast 提示「请输入正确的密码」
- 返回上一页：回到邮箱输入页，邮箱**不清空**

---

## 六、手机号登录流程

```
手机号输入页 → [已注册] → 密码输入页 → 登录成功
                ↓ [未注册]
             提示「去邮箱注册」
```

### 手机号输入规则
- 拉起数字键盘，支持选择区号
- 最大 **11 个字符**，超出不显示
- 格式与所选区号不一致：提示「请输入正确的手机号码」
- 返回上一页：输入框清空

---

## 七、找回密码流程

### 邮箱找回密码
```
邮箱登录页点击「忘记密码」
→ 图形验证码
→ 邮箱验证码页（60s 倒计时，验证码 5 分钟有效）
→ 重置密码页
→ 密码重置成功并登录
```

### 手机号找回密码
```
手机号登录页点击「忘记密码」
→ 图形验证码
→ 短信验证码页（60s 倒计时，验证码 5 分钟有效）
→ 重置密码页
→ 密码重置成功并登录
```

### 验证码规则（邮箱/短信）
| 规则项 | 说明 |
|--------|------|
| 倒计时 | 60s，倒计时内「重新发送」按钮置灰 |
| 倒计时结束可重发 | 点击「重新发送」发送新验证码，重新开始倒计时 |
| 验证码 5 分钟后过期 | 过期后点击「重新发送」需重新通过图形验证码 |
| 验证码错误 | 输入框抖动变红，清空输入框，toast 提示「验证码错误，请重试」|
| 返回上一页 | 验证码输入框清空 |
| 再次进入验证码页 | 倒计时重新开始（60s 内返回再进入也会重发）|

## 八、邮箱注册流程

```
邮箱输入页 → 图形验证码 → 邮箱验证码（60s 倒计时）→ 设置密码页
→ 设置名称页 → 生日校验页 → （可选）接受个性化邮件 → 注册并登录
```

### 各步骤说明

**邮箱输入页：**
- 已注册邮箱 → 点击下一步提示「邮箱已注册」
- 不允许输入空格、中文，最大 254 个字符

**设置名称页：**
- 输入框默认显示系统生成名称（如 `viaim1234`）
- 名称最大 **10 个字符**，超出不允许继续输入

**生日校验页：**
| 年龄 | 行为 |
|------|------|
| **未满 14 岁** | 提示「超过14岁才可以注册 viaim 账号」，不可注册 |
| **满 14 岁且不足 17 岁** | 弹出确认声明弹窗，需勾选两个选项后才可注册 |
| **满 17 岁及以上** | 直接可注册 |

**接受个性化邮件（可选）：**
- 注册时勾选 → 账号设置中「接受个性化邮件」开关开启
- 注册时不勾选 → 对应开关关闭

## 九、Google 登录流程

| 场景 | 行为 |
|------|------|
| 三方已注册（邮箱有无均可）| 直接登录成功 |
| 三方未注册，邮箱未注册 | **授权后直接注册并登录** |
| 三方未注册，邮箱已注册（未绑定三方）| **邮箱验证码验证通过后绑定并登录** |
| 三方未注册，邮箱已注册（已绑定 Apple）| 进入安全验证页，邮箱验证码验证通过后 Google + Apple 均绑定并登录 |
| 三方未注册，邮箱已注册（已绑定其他 Google）| **阻断**，提示「当前谷歌账号邮箱已绑定其他账号，请直接用邮箱登录」|
| Google 已绑定邮箱但无密码 | 使用邮箱登录无特殊提示，进入密码输入页后输入密码提示错误；可走**找回密码流程**设置密码后登录 |
| 点击取消/关闭授权弹窗 | 返回登录页 |

---

## 十、Apple 登录流程

| 场景 | 行为 |
|------|------|
| 三方已注册（含隐藏/共享邮箱）| 直接登录成功 |
| 三方未注册，选择「隐藏邮件」| **授权后直接注册并登录**，邮箱空，Apple 已绑定 |
| 三方未注册，选择「共享邮件」，邮箱未注册 | **授权后直接注册并登录**，邮箱显示共享邮箱，Apple 已绑定 |
| 三方未注册，选择「共享邮件」，邮箱已注册（未绑定三方）| **邮箱验证码验证通过后绑定并登录** |
| 三方未注册，选择「共享邮件」，邮箱已注册（已绑定 Apple）| **阻断**，提示「当前苹果账号邮箱已绑定其他账号，请直接用邮箱登录」|
| 三方未注册，选择「共享邮件」，邮箱已注册（已绑定 Google）| 进入安全验证，邮箱验证码验证通过后 Google + Apple 均绑定并登录 |
| Apple 已绑定邮箱但无密码 | 使用邮箱登录无特殊提示，进入密码输入页后输入密码提示错误；可走**找回密码流程**设置密码后登录 |
| 点击取消/关闭授权弹窗 | 返回登录页 |

> **Apple 登录平台说明：**
> - iOS：显示 Apple 登录按钮
> - Android：若账号已绑定 Apple，账号设置中显示「Apple 已绑定」，可解绑；但 Android **不显示**「Apple 未绑定」入口（即不支持在 Android 端发起绑定 Apple）

## 十一、首次登录引导

### 新注册用户

| 规则 | 说明 |
|------|------|
| Record 列表 | 自动创建示例文件《Welcome to viaim》.mp3，创建时间为登录时间 |
| Space Tab | 自动创建演示 Space「Welcome to viaim」，含示例 .mp3（快捷方式）和 .pdf（文档） |
| 支持编辑和删除 | 示例文件/演示 Space 支持编辑和删除操作 |
| 删除后不重建 | 删除后再次登录不会重新创建 |

### 登录同步行为

| 场景 | 行为 |
|------|------|
| 退出重新登录 | 显示"正在同步记录"页面，同步成功后展示记录 |
| 登录状态下升级安装 App | 不显示同步页面，直接展示记录列表 |

> 老版本用户首次登录 2.0 的迁移规则 → 详见 `domain/version-migration.md`

## 十二、测试关注点

- 首次安装弹出提醒弹窗，不同意则退出 App
- 未勾选协议点击任意登录/注册按钮：条款抖动，不跳转
- 密码规则：8位+大小写+数字，三条实时校验
- 图形验证码：5分钟过期、1分钟20次刷新上限
- 验证码：60s 倒计时；5分钟过期后重发需再次图形验证
- 邮箱登录返回：邮箱不清空；验证码页返回：验证码清空
- 手机号登录未注册：提示去邮箱注册（非去注册）
- 生日校验：14岁以下阻断；14-17岁需确认声明；17岁以上直接注册
- 注册名称默认值（如 viaim1234）、最大10字符
- Google/Apple 三方登录各场景的分流逻辑（已存在/新用户/邮箱冲突）
- Apple 隐藏邮件 vs 共享邮件的账号状态差异
- Android 不显示 Apple 未绑定入口
- 三方新用户（邮箱未注册）：授权后直接注册，不走设置密码/名称/生日校验流程
- 三方新用户（邮箱已注册未绑定）：邮箱验证码验证后绑定并登录（非密码验证）
- 已绑定邮箱但无密码的账号：邮箱登录无特殊提示，密码输入提示错误，需走找回密码流程
- **首次登录**：新用户自动创建示例文件和演示 Space，删除后不重建
- **登录同步**：退出重登显示同步页面；升级安装不显示同步

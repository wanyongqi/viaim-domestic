# 左边栏与文件列表功能规格（Web海外版基线）

> 版本：V2.0.2 | 最后更新：2026-04

---

## 一、左侧边栏导航

### 1.1 固定分类导航
- **全部文件**：展示所有非回收站记录，按创建时间降序排列。
- **我的录音**：仅展示来源为耳机的录音（包含闪录）。
- **上传文件**：仅展示用户手动上传的音频与文档。
- **未分组**：仅展示未加入任何分组的记录。
- **回收站**：展示已删除的记录，且卡片无左滑操作。

### 1.2 分组管理（Space）
- **自定义分组**：点击任意自定义分组，右侧列表展示该分组下的所有记录（含快捷方式）。
- **展开与折叠**：点击分组标题区域，可正常展开/折叠子分组列表。
- **超长边界值**：分组名称超过侧边栏宽度时，尾部显示省略号（...）。
- **Hover操作**：鼠标悬停在分组项上，右侧出现「修改」等操作按钮。
- **滚动边界**：分组数量过多超出屏幕高度时，左侧栏可独立无缝滚动。

### 1.3 底部入口
- **容量展示**：展示当前已用容量与总容量进度条（仅国内版有总容量限制，海外版展示已用容量）。
- **下载客户端**：点击跳转至正确的客户端下载页面或弹出下载引导。

## 二、文件列表展示与交互

### 2.1 列表卡片信息展示
- **正常录音卡片**：包含标题、概览/转写首行、录音类型图标、时间/时长。
- **上传文件卡片**：标题为文件名+后缀，右上角展示格式标签（如MP3/PDF）。
- **超长显示**：标题或概览超过一行时，尾部显示省略号。
- **AI生成状态标签**：根据记录实际状态，正确展示「转写」「摘要」「待办」「思维导图」等标签。
- **未读红点**：本端生成且未查看的记录，卡片标题左侧显示红点。

### 2.2 列表状态与加载
- **空状态**：当前分类下无记录时，展示对应缺省图与文案。
- **分页加载**：滚动到底部时自动加载下一页数据，加载中显示Loading状态。
- **异常加载**：断网/弱网下加载列表，提示网络异常，支持点击重试或保持原有缓存数据。

### 2.3 排序功能
- **弹窗交互**：点击弹出下拉弹窗，点击选项直接应用并收起下拉弹窗。
- **互斥可选项（8个）**：创建时间降序/升序、修改时间降序/升序、名称降序/升序、录音时长降序/升序（仅在【全部录音】/【我的录音】中有），默认顺序为创建时间降序。
- **可勾选项**：按照文件类型排序；勾选后，先根据“设备录音→上传音频→上传文档→上传视频→上传网页”的顺序，再根据上方互斥可选项选中的顺序排序（【全部录音】/【我的录音】中没有该项）。
- **同步规则**：排序设置在 Web 端同步，刷新 Web 排序仍然生效；不同 Space 内的排序同步生效。

### 2.4 筛选功能
- **弹窗交互**：点击弹出筛选弹窗，点击筛选项时弹窗**不会收起**；点击弹窗以外区域收起弹窗；点击【查看结果】收起弹窗并应用选择的结果。
- **可选项**：日期、文件类型（互斥单选）。
- **日期筛选交互**：
  - 当天日期亮起；有记录的日期亮起标记，没有的仍然可交互但需突出视觉不一样（按创建时间）。
  - 点击 `<` / `>` 切换月份；点击 `<<` / `>>` 切换年份。
  - **多日期预选中**：点击日期选中后，光标悬浮到其他日期，距离原日期之间的日期上出现覆盖遮罩，代表多日期预选中。再次点击确定起止日期，高亮显示。
  - **快捷选择**：点击【今天】、【最近7天】、【最近30天】，下方的日历视图会跟着选中对应范围。
  - **清除**：选择了至少一个日期或范围后，【清除】按钮亮起；点击清除日期选择，日历展示的月份不变。
- **应用状态**：筛选应用后，按钮展示应用状态。此时若全选文件，实际选中的只有筛选结果中的文件。
- **空结果**：筛选后无结果时，展示无结果界面，文件详情区域展示的记录不变。
- **状态保留**：刷新页面时，筛选状态保留；分组切换恢复默认（不应用筛选）。

### 2.5 导出与拖拽功能
- **单文件导出**：在列表导出单个文件时，**不打包为 zip 格式**。
- **文件拖拽**：
  - 按住拖拽文件卡片，可以添加到自定义分组、回收站（需二次确认）。
  - 【回收站】中的记录无法拖拽操作。
  - 拖拽到自定义分组上方松手后加入对应分组，弹出 Toast：“已加入分组<分组名称>”；若文件已在该分组中，弹出 Toast：“文件已在该分组中”。

## 三、异常与边界场景

- **解析失败记录**：卡片红框显示，仅保留清除按钮，不可进入详情。
- **上云失败记录**：显示红色云图标，支持点击重试。
- **浏览器缩放**：缩小浏览器窗口宽度，左侧栏与文件列表布局自适应，不发生重叠或错位。
